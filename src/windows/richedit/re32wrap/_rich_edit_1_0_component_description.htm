Rich Edit 1.0 Component Description

This component contains an emulation of Rich Edit 1.0 (Riched32.dll), an ANSI dynamic-link library (DLL) for text editing and display. Use of this component has been deprecated; new applications should use Rich Edit 3.0 (Riched20.dll) or later.

Component Configuration

There are no configuration requirements for this component.
