MS Candidate UI Component Description

This component contains a Component Object Model (COM) server that provides interfaces for text services to choose alternatives other than the one deemed most likely by the input engine. The candidate user interface (UI) hosts a candidate list that presents alternatives to the user as well as other functions related to correction. This is a required component for East Asian Input Method Editor (IME) components.

Component Configuration

There are no configuration requirements for this component.
