AIMM1.2 Component Description

Active Input Method Manager (AIMM) 1.2 is a Component Object Model (COM) server that provides Input Method Manager (IMM) East Asian Input Method Editor (IME) services to implementing clients, such as Microsoft Internet Explorer, Microsoft Word, or other AIMM-aware applications, on non-East Asian versions of Microsoft Windows. Also as a compatibility layer, it provides modal input from Microsoft Windows Text Services Framework-based services to implementing clients on all versions of Windows.

Component Configuration

There are no configuration requirements for this component.
