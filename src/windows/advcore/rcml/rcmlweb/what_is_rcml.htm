RCML - Product Definition
overview

RCML is a new Windows resource file format that is designed to provide access to more Win32 UI features than existing RC files and solve a number of problems with existing dialog boxes and the existing resource format, most notable with localization and UI design. In addition to the new RCML file format, there is an enhanced version of the DialogBox API that works hand in hand with the RCML file to make use of the enhanced data it contains.

 schedule and deliverables

RCML  is scheduled to ship one year after Windows 2000 Professional.
