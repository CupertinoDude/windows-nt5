Installation

There are 3 parts to installing the Voice Demos; you need SAPI, CICERO, and the demo apps. I have this on a Windows 2000 Server, 2195 build.

  1. Install SAPI.
  2. Install Cicero
  3. Install a Cicero enabled wordpad

You should now have two new Apps on the start menu. Start with the "Cicero Wordpad". This is the MFC sample application from MSDN, with 2 lines of code changed - which make it FULLY Cicero enabled.

When you put focus on Wordpad, you should see a small toolbar much like this:

The "Mic" button indicates that dictation is on. When you start speaking, you'll see ............ when there is a suitable pause, this will be converted to text.

If you have a wacom tablet, then you need to install some additional drivers.

  1. Wacom drivers for Win2K
  2. Hand writing tip thingy from Cicero.
