RCML  .  MS Paint Demo

This version of  MSPaint has the dialogs created using RCML.   It is a typical MFC application and it's sources are checked in the RCML tree.
You can play with the dialogs  in the Image menu and modify their appearance by editing the .xml files with notepad or any other text editor.
