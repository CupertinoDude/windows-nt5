RCML  .  Weekly Drops

Here is the latest drop of the RCML build files.  This is a work in progress, so if you want stable builds, please use our M0 build (found in the "builds" folder) or the latest.idw build.  

To use these RCML builds, either run the installer for that build (if there is one there) or create a folder and copy the files from the latest.idw into that folder.

Once you have copied the files to your machineDrag the .XML files onto RCMLGen.EXE, or simply run RCMLGen.EXE and then generate RCML files from existing applications! There is a tutorial on our site to get you going.

Click here if you want to play with an MSDev project converted to use RCML. Copy all the files locally to your machine, and have fun!
