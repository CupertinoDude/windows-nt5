GetGadget

Core

GetGadget retrieves the Gadget that has the specified relationship to the specified Gadget.

HGADGET GetGadget(
    HGADGET hgad,               // Handle of Gadget 
    UINT nCmd)                  // Relationship
);

Parameters

hgad
[in] Handle of Gadget
nCmd
[in] nFlags can be a combination of the following flags:

Value Meaning
GG_PARENT Return the parent of the specified Gadget.
GG_NEXT Return the next sibling behind the specified Gadget.
GG_PREV Return the previous sibling before the specified Gadget.
GG_TOPCHILD Return the Gadget's top z-ordered child.
GG_BOTTOMCHILD Return the Gadget's bottom z-ordered child.

Return Values

Related Gadget or NULL for none.

See Also

SetGadgetOrder, SetGadgetParent
