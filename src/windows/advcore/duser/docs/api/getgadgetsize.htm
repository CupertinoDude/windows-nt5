GetGadgetSize

Core

GetGadgetSize is a high-performance mechanism of retreiving the Gadget's logical size.

BOOL GetGadgetSize(
    HGADGET hgad,               // Handle of Gadget
    SIZE * psizeLogicalPxl      // Size in logical pixels
);

Parameters

hgad
[in] Handle of Gadget
psizeLogicalPxl
[out] Size in logical pixels

Return Values

Successfully returned size in logical pixels.

GetGadgetRect, SetGadgetRect, Gadget Styles
