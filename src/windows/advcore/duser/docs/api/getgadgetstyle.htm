GetGadgetStyle

Core

GetGadgetStyle returns the current style of the given Gadget.

UINT GetGadgetStyle(
    HGADGET hgad                // Handle of Gadget
);

Parameters

hgad
[in] Handle of Gadget

Return Values

Current style of Gadget

Remarks

For a list of Gadget styles, see SetGadgetStyle.
