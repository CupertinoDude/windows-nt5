RegisterGadgetMessageString

Core

RegisterGadgetMessageString defines a new private Gadget message that is guaranteed to be unique throughout the process. This MSGID can be used when calling SendGadgetMessage or PostGadgetMessage. The MSGID is only valid for the lifetime of the process.

MSGID RegisterGadgetMessageString(
    LPCWSTR pszName             // Unique string ID of message to register
);

Parameters

pszName
[in] Unique string ID of message to register

Return Values

ID of new message or 0 if failed.

Remarks

See RegisterGadgetMessage for more information about MSGID's.
