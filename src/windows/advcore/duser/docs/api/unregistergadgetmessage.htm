UnregisterGadgetMessage

Core

UnregisterGadgetMessage decreases the reference count of a private message by one. When the reference count reaches 0, resources allocated to store information about that private message are released, and the MSGID is no longer valid.

MSGID UnregisterGadgetMessage(
    const GUID * pguid)         // Unique GUID of message to unregister
);

Parameters

pguid
[in] Unique GUID of message to unregister

Return Values

Message was successfully unregistered.

Remarks

See RegisterGadgetMessage for more information about MSGID's.
