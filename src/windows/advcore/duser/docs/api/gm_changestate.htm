GM_CHANGESTATE

Core

GM_CHANGESTATE is sent by the Core when a state change occurs in the Gadget.  

GMSG_STATE : GMSG
{
    UINT        nCode;          // Change command
    HGADGET     hgadSet;        // Gadget that is receiving the "state"
    HGADGET     hgadLost;       // Gadget that is loosing the "state"
    UINT        nCmd;           // Action that occurred
};

Parameters

nCode
nCode specifies the type of change that occurred:

 

Value Change
GSTATE_FOCUS Keyboard input focus has been set or lost from the window. 
GSTATE_ACTIVE The "active" state has been set or lost from the window.
GSTATE_CAPTURE Mouse input has been set or lost from the window.
hgadSet
Gadget that is receiving the "state"
hgadLost
Gadget that is loosing the "state"
nCmd
nCmd specifies whether the state was given or lost: 

 

Value Change
GSC_SET State was given.
GSC_LOST State was lost.

Return Values

Gadget with keyboard focus.
