Microsoft Korean IME Core Component Description

This component includes the Korean Input Method Editor (IME) and Korean keyboard text input processor (TIP) for the text services framework, which allows users to input Korean text using the keyboard.

Component Configuration

There are no configuration requirements for this component.

Special Notes

To enable handwriting functionality in IME through the handwriting IME pad applet, you must install the following components:

  • Microsoft IME pad
  • Korean handwriting recognizer
  • Microsoft Korean IME handwriting applet
