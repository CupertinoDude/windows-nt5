MSMQ Triggers Component Description

The Message Queuing (MSMQ) Triggers component allows you to associate the arrival of each incoming message at a queue with a response that depends on the contents of the message and may invoke either a Component Object Model (COM) component or a stand-alone executable program. By using Message Queuing Triggers, specific behavior can be invoked at the queue level.

Component Configuration

There are no configuration requirements for this component.

Special Notes

In order to install the MSMQ Triggers component, you must also install the MSMQ Admin component.
