MSMQ Cluster Resource Component Description

This component defines the functionality required to support clustering of Message Queuing computers.

The Message Queuing service, and Message Queuing triggers, can be implemented in clusters. A cluster is a set of computers, known as nodes or hosts, which work together under a common identity to provide a service. All clusters provide load balancing, increased availability, and scalability.

Component Configuration

There are no configuration requirements for this component.

Special Notes

In order to install the MSMQ Cluster Resource component, you must also install the MSMQ Core component.
