MSMQ COM API Component Description

This component includes the Message Queuing Component Object Model (COM) objects, implemented as part of the Message Queuing Software Development Kit (SDK).

Message Queuing provides COM components that support queue lookup, queue management, message management, queue administration, and transaction support. As a group, these components provide most of the same functionality as the Message Queuing application programming interface (API) functions.

Component Configuration

There are no configuration requirements for this component. 

Special Notes

In order to install MSMQ COM API component, you must also install the MSMQ C API component.
