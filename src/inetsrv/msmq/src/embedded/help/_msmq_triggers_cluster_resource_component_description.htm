MSMQ Triggers Cluster Resource Component Description

The Message Queuing (MSMQ) Triggers Cluster Resource component allows you to create triggers for MSMQ clusters.

Message Queuing Triggers associate the arrival of each incoming message at a queue with a response that depends on the contents of the message and may invoke either a Component Object Model (COM) component or a stand-alone executable program.

Message Queuing Triggers can be implemented in clusters. A cluster is a set of computers, known as nodes or hosts, which work together under a common identity to provide a service.

Component Configuration

There are no configuration requirements for this component.

Special Notes

To install the MSMQ Triggers Cluster Resource component, you must also install the MSMQ Triggers component.
