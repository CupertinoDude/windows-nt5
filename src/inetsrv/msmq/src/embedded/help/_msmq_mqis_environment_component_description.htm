MSMQ MQIS Environment Component Description

The Message Queuing (MSMQ) Message Queue Information Store (MQIS) Environment component allows users to connect to Message Queuing 1.0 supporting servers.

This component allows users to query queues located on previous MSMQ computers.

Component Configuration

This component requires a Windows NT 4.0 or later primary enterprise controller (PEC) enterprise.

Special Notes

To install the MSMQ MQIS Environment component, you must also install the MSMQ Core component.

You cannot install the MSMQ MQIS Environment component, if you install the MSMQ AD Environment component.
