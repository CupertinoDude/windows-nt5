ADSI Changes for IIS 5.1

This section contains information about the changes and additions in the ADSI provider for and IIS 5.1.

  • IIS Performance Features: Describes new performance features for IIS 5.1.
  • ADSI Properties Removed from IIS 5.1: Lists any properties that were supported in previous versions, but which are not valid for IIS 5.1.
  • ADSI Properties Added to IIS 5.1: Provides a list of properties new to IIS 5.1, and links to each property's full reference.
  • ADSI Property Key Type Changes for IIS 5.1: Lists any properties whose key type has changed for IIS 5.1.
  • ADSI Properties Changed in IIS 5.1: Lists any properties that have different default behavior for IIS 5.1.
