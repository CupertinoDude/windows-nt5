Status

The Status method of the IIsWebServer object returns an integer that indicates the current status of the server. The possible values and their meaning are listed in the following table.

Values Description
1 Starting
2 Started
3 Stopping
4 Stopped
5 Pausing
6 Paused
7 Continuing

Syntax

IIsWebServer.Status

 

See Also
