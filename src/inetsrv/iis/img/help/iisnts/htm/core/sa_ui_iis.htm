Internet Information Services

You can configure general settings for IIS by right-clicking on Internet Information Services in the navigation pane of the MMC. These general settings include:

Internet Information Services Property Sheet containing properties for

  • bandwidth throttling
  • MIME mapping. 

IIS Server Extensions containing properties for

  • Office Web Server server extensions
  • FrontPage server extensions
  • Performance
  • Client scripting
  • Mail settings
  • Permissions

You can also configure your Backup Configuration procedures by selecting this option from the menu after right-clicking on Internet Information Services in the navigation pane.
