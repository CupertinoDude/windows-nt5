Display Administrative Node (dispnode)

Summary:

Displays the relevant fields for a particular administrative node in the tree.

Usage:

dispnode<--ADSPath|-a ADSPATH>
<--help|-h>

Parameters:

ADSPATHFull ADSI path to the node to be displayed.

Example:

Dispnode -a IIS://LocalHost/w3svc
