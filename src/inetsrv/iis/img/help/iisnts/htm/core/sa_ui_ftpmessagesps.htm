Messages Property Sheet

Use this property sheet to create your own messages that will be displayed to browsers that connect to your site.

Welcome

Displays this text to clients when they first connect to the FTP server. The message is blank by default.

Exit

Displays this text to clients when they log off the FTP server. The message is blank by default.

Maximum Connections

Displays this text to clients that try to connect when the FTP service already has the maximum number of client connections allowed. The message is blank by default.
