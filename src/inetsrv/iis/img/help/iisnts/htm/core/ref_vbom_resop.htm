Response Properties

The Response Object object has the following properties:

  • Buffer
  • CacheControl
  • Charset
  • CodePage
  • ContentType
  • Expires
  • ExpiresAbsolute
  • IsClientConnected
  • LCID
  • PICS
  • Status
