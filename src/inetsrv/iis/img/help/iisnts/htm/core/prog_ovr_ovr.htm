Overview of Programmatic Administration

Programmatic administration allows developers and administrators to automate access to IIS configuration parameter values stored in the metabase. 

This section contains:

  • Overview of Programmatic Administration: Introductory description of programmatic administration.
  • Using the WMI Interface: Explains how to use Windows Management Instrumenation (WMI) to query and manipulate the IIS metabase.
  • Using IIS Admin Objects: Explains how to use Active Directory Services Interface (ADSI) to manipulate the metabase.
  • Development Choices for Programmatic Administration: Reference pages for WMI and ADSI.

This section does not contain information on:

  • Metabase concepts: Metabase: Describes the IIS metabase.
