Configuring IIS for Passport Authentication

This topic is under construction.

This topic describes how to enable and configure Passport authentication on your IIS server through the IIS snap-in. For information on configuring Passport authentication programmatically, see Programmatic Considerations for Passport Authentication

Passport is not a highly secure protocol like Digest authentication. Passport uses cookies. To avoid replay attacks, use Passport authentication over a Setting Up SSL on Your Server. For more information about secure protocols, see About Authentication.

The Microsoft Passport Business Services home page offers comprehensive developer and technical support resources to help businesses implement Passport services. For more information on licensing and implementing Passport, visit http://www.passport.com/business/.
