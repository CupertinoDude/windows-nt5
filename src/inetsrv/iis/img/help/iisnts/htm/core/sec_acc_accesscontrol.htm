Access Control

You can control which users and which computers are allowed to access to your Web server and its resources. You can use both NTFS and Web server security features to assign users specific access rights to directories and files.

This section contains:

  • About Access Control: Overview information about restricting access to files and directories.
  • Using NTFS Permissions to Secure IIS: Describes how to use NTFS Permissions to secure your Web server.
  • Granting and Denying Access to Computers: Preventing specific computers, groups of computers, or entire networks from accessing your Web content.
  • Setting Web Server Permissions: Setting Web and FTP content restrictions that apply to all users.
  • Controlling Database Access: Guidelines for protecting your database files from unauthorized access.

This section does not contain information on:

  • Configuring content rating features to warn users about objectionable Web content; see Using Content Ratings.
  • Configuring your Web server's Secure Sockets Layer (SSL) security features; see Setting Up SSL on Your Server.
