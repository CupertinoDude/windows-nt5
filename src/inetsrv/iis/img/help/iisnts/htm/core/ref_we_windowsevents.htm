Windows Events for IIS

IIS generates events in the Event Viewer so that you can verify the performance of IIS. Event Viewer tracks (in order of importance) error events, warning events, and informational events. The logs in Event Viewer provide an audited record of all Windows services and of processes such as logon times and connection times.

This section includes:

  • About Windows Events: Conceptual topics related to Windows Events.
  • Web Services Events: List of events generated from Web services.
  • FTP Services Events: List of events generated from FTP services.
  • Active Server Pages Events: List of events generated from ASP.

This section does not contain information on:

  • Error messages displayed in the browser of a client; see Custom Error Messages.
  • Logging client requests; see Logging Site Activity.
