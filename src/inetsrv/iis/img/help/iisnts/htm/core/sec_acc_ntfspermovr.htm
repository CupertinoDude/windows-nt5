Using NTFS Permissions to Secure IIS

With NTFS you can limit access to your Web server's files and directories. You can configure the access level granted to a particular user account or user group for all the files and directories on your server.

This section contains the following topics:

  • Securing Your Files with NTFS: Procedural information about configuring your server to use the NTFS security features.
  • Setting NTFS Permissions for a Directory or File: Procedural information about configuring NTFS permissions, which define the level of file or directory access assigned to a Windows user account.
