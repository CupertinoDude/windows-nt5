Getting Started with Internet Information Services 6.0

Internet Information Services (IIS) 6.0 is a full-featured Web hosting server that leverages interactive applications. IIS 6.0 supports the latest Internet standards, which allows your business to take advantage of the full scope of Internet technologies. IIS 6.0 features increase manageability, reliability, availability, security, and performance of your Web server. You can administer IIS using the user interface, known as the IIS snap-in, or through command line scripts. IIS is the Windows Whistler Web service that makes it easy to publish information on your intranet or on the Internet.
