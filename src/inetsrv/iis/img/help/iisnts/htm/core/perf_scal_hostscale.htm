Host Scaling (Administering Large Quantities of Sites)

This topic is under construction

Microsoft.com currently houses information on the Scalable Hosting Solution, an IIS virtual hosting ISAPI filter sample application. The Scalable Hosting Solution (SHS) sample application enhances the ability of IIS to host thousands of static-content sites using an ISAPI filter that provides an alternate approach to configuring individual sites in the metabase.
