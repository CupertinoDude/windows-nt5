Internet Information Services Snap-in User Interface Reference

This reference describes in detail the configurable settings available on the IIS snap-in. Use this documentation in conjunction with configuring IIS through the snap-in or programmatically, and as a means to distribute your specific administrative procedures. 

You can access the snap-in through the Computer Management console (MMC) under the Administrative Tools section of the Control Panel. When installed, Internet Information Services is listed in the navigation pane of the MMC, along with all the servers you have installed. To view the property sheets, right-click on any server or site, and select Properties.

The default selections are:

Internet Information Services which contain properties for Server Extensions, bandwidth throttling, and MIME mapping. You can also configure your backup and restore procedures from here.

Web Site Property Sheets properties including custom errors, directory security, documents, HTTP headers, the home directory, site operators, performance, server extensions, and the Web site itself.

FTP Site Property Sheets properties including directory security, the FTP site itself, home directory, messages, and security accounts.

The Administrative site allows the same configuration options as the default Web site. The Default SMTP Virtual Server is not described in this reference documentation. For more information about SMTP, see the Microsoft SMTP Service online help.
