Using the IIS WMI Interface

Windows Management Instrumentation (WMI) allows Windows administrators to manage IIS programmatically using the IIS WMI provider. The provider exposes programming interfaces that query and configure the metabase.

  • About the IIS WMI Interface   Introduces WMI, the IIS WMI provider, and how they are used.
  • IIS WMI Object Hierarchy   Describes the WMI schema, and how classes and instances correspond to the IIS metabase schema.
  • IIS WMI Objects   Defines the WMI classes.
  • Manipulating the Metabase Using IIS WMI Provider   Describes metabase management tasks and provides scripting examples.
