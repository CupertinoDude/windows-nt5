Metabase Structure

The metabase is a hierarchical repository for most IIS configuration values, while a few IIS configuration values are stored in the Registry.

This section contains:

  • Metabase Configuration File  Describes the hierarchical structure of the MetaBase.XML file.
  • Metabase Schema  Explanation of what the metabase schema is used for, and the structure of the MBSchema.XML file.
  • Property Inheritance  Explains how property values are inherited from higher nodes if not specified otherwise.
