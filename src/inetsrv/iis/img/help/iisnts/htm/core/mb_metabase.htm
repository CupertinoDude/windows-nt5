Metabase

The metabase is a hierarchical repository for most IIS configuration values, while a few IIS configuration values are stored in the Registry.

This section contains:

  • About the Metabase: Overview of  the IIS metabase.
  • Metabase Structure: Describes the hierarchical structure of the Metabase, inheritance of property values. Also describes the metabase schema.
  • Configuring the Metabase:  Discusses the various methods for configuring the metabase. There is a discussion of history files, editing the MetaBase.xml file while IIS is running, as well as an explanation of extending the XML metabase schema.
  • Metabase Security:  Describes file level security of the metabase files, encrypted metabase properties, and auditing. Also discusses programmatic access to metabase nodes and renaming or relocating MetaBase.xml.
  • Metabase Reliability:  Discuses backing up and restoring the metabase.
  • Metabase Import and Export:  Describes Site cloning, application cloning, and creating a metabase template to import/merge from other IIS computers.
  • Metabase Changes for IIS 6.0:  Overview of the metabase properties that have changed in IIS 6.0.
  • Metabase Scenarios:  Scenarios that illustrate metabase functionality.
