Logging Properties Reference

This topic lists definitions of logging properties and their values for the W3C Extended logging format. It is divided into these sections:

  • W3C Extended Logging Definitions: General properties available to the W3C Extended log file format.
  • Process Accounting Logging Definitions: Additional properties for process accounting that appear in the W3C Extended log files.
  • Microsoft IIS Log Definitions: General properties available to the Microsoft IIS log file format.
  • NCSA Common Log File Format: General properties available to the NCSA Common log file format.

For more information on the W3C Extended log format, see the World Wide Web Consortium Web site.
