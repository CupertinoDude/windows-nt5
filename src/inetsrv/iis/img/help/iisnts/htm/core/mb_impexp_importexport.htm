Metabase Import and Export

Describes Site cloning, application cloning, and creating a metabase template to import/merge from other IIS computers.

This section contains the following topics:

  • Moving Sites to Another Computer  Explains how to clone a site and then export the site to another computer.
  • Moving Applications to Another Computer   Outlines how to move applications to another computer.
  • Metabase Template    Overview of the metabase template.
  • Metabase Recovery   Explains how to recover a section of the metabase without the need to restore the entire metabase.
