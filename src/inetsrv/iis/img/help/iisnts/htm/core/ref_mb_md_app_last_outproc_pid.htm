MD_APP_LAST_OUTPROC_PID

In some cases when an application is moved from out of process to in-process, the application's corresponding package may not be deleted by IIS. In this case, IIS stores the application package ID with the value of this property so that if the application is later moved out of process, IIS can use the existing package in MTS/COM+.

Caution    This setting is used internally by IIS and should not be modified by administrators.
