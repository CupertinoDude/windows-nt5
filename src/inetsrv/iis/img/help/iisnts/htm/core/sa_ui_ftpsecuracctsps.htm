Security Accounts Property Sheet

Use this property sheet to control who can use your server and specify the account used for anonymous client requests to log on to the computer.

Anonymous Connections

To allow clients using the user name "anonymous" to log on to your FTP server, select this option.

To add a Windows user account to be used for anonymous connections, click Add.

To remove a Windows user account, select it and click Remove.

Use this property sheet to define which Windows User Accounts have Operator privileges to this site.

FTP Site Operators

To add a user account to the list of Operators, click Add.

To remove the currently selected user account, click Remove.

An operator can carry out a limited set of FTP site administrative functions, including setting FTP server access permissions.

Users with Operator privileges cannot carry out strictly administrative functions, such as changing the identification of FTP sites.
