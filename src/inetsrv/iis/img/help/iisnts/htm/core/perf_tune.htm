Performance Tuning

This section describes how to monitor your Internet Information Services (IIS) server and how to adjust settings to increase overall performance.

This section contains the following topics:

  • About Performance Tuning: This topic describes a general methodology for testing and tuning your IIS Web server. This topic also describes the IIS Quality of Service (QoS) performance features.
  • Disk Optimization: This topic describes how to monitor and tune IIS for improved disk access.
  • Performance Settings: This topic describes specific registry and metabase settings that you can adjust to improve IIS performance.
  • Performance Monitoring: This topic describes the tools and counters used to monitor server performance.
  • Scalability: This topic describes different methods of hardware scaling to improve IIS performance.
  • Memory Management: This topic describes how to balancing memory requirements against disk access speed.
  • Network Capacity: This topic describes how to plan network capacity and maximizing connection performance.

This section does not include information on:

  • Programmatic administration; see Programmatic Administration.
  • Performance effects of site content choices; see Configuring Applications.
  • Tracking Web site traffic; see Logging Site Activity.
