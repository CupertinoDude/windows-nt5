Ratings Property Sheet

Use this property sheet to configure your Web server's content rating features for a particular Web site, directory, or files.

Your Web server's Platform for Internet Content Selection (PICS) system uses a rating system developed by the Recreational Software Advisory Council (RSAC), which rates content according to levels of violence, nudity, sex, and offensive language.

To apply content ratings to your Web content, select the Enable Ratings for this resource check box.

To set content ratings, in the Categories box click a ratings category (Language, Nudity, Sex, or Violence). To set the level of potentially objectionable material for the category, use the rating slider. Each setting displays a description of the rating level.

To identify the person who rated the content, enter an e-mail address in the Email name text box.

To set an expiration date for rated content, click the button to the right of the Rating expires box. A calendar control will appear where you can specify an expiration date.
