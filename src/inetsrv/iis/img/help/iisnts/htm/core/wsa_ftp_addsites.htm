Adding Sites

You can add new FTP sites to a computer by launching the site wizard. The FTP service must be installed before FTP sites can be created.

To add an FTP site
  1. In the IIS snap-in, select the computer or a site, and click Action.
  2. Select New, and then FTP Site to launch the site wizard.
  3. Follow the on-screen directions to assign identification information to your new site. You must provide a unique IP address/port address combination and the home directory path.

    Note    In the site wizard, All Unassigned refers to IP addresses that are assigned to a computer but not assigned to a specific site. Only one site can be set to use unassigned IP addresses.
