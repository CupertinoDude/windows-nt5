Internet Information Services Property Sheet

Use this property sheet to access all the properties for any installed service on this computer, such as Web or FTP. You can also set the bandwidth used by all services and to configure MIME types. These default values can be changed on an individual basis for each site if desired.

Master Properties

Select the services available to edit any associated property with the site.

Enable Bandwidth Throttling

Select this option to limit the bandwidth used by Web and FTP services on your computer.

Limiting the bandwidth is especially useful if your network card has multiple purposes, such as e-mail and remote logons. Bandwidth throttling limits only the bandwidth used by static HTML files.

MIME Map

Select the File Types button to configure Multipurpose Internet Mail Extensions (MIME) mappings. These mappings set the various file types that the Web service returns to browsers.
