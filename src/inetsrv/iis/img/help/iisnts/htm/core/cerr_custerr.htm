Custom Error Messages

You can configure IIS to send custom error messages instead of the default HTTP 1.1 error messages. These custom error messages can be mapped to a file name or to a URL.

This section includes:

  • About Custom Error Messages: Conceptual information about default and custom HTTP 1.1 errors.
  • Adding a Custom Error Message: Procedural information about adding custom error messages to your Web site.
  • Enabling Detailed Custom Error Messages: Procedural information about using the detailed (or "friendly") HTTP 1.1 errors that are included with IIS.
  • Enabling ASP Error Processing: Conceptual information about using the 500-100.asp custom error file to process ASP errors.

This section does not contain information on:

  • Web site content; see Web Site Management.
  • Windows Events; see Windows Events for IIS.
