Sample Administration Scripts

Administration Scripts are tools that administrators can use to change Web server settings by using a command-line scripting utility, Cscript.exe.

This section includes:

  • About Administration Scripts: Explains the functionality of Administration Scripts.
  • Scripts Documentation: Describes each script, with usage and parameter information.

This section does not contain information on:

  • Administrator's Reference Information; see Administrator's Reference.
