Administrator's Reference

This section contains all the reference material for IIS and ASP:

  • Registry describes the stored values that define the working environment for the Windows operating system and any services installed on a computer running Windows
  • Metabase Property Reference describes the properties of the metabase that are configurable from scripts
  • Windows Events for IIS describes windows events, Web services events, FTP services events,  and Active Server Pages events
  • Glossary describes terms specific to the operating system and IIS, as well as definitions of relevant concepts.
