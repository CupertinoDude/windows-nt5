Installing the FTP Service

The FTP service is installed by default when IIS 6.0 is installed. A default FTP site is created at x:\Inetpub\Ftproot when the FTP service is installed, where x: represents the drive letter where IIS is installed. If IIS is installed and the FTP service is un-installed, you can reinstall the FTP service by following the procedure below.

To install FTP from the IIS optional components
  1. Click Start, point to Settings, click Control Panel, and start the Add/Remove Programs application.
  2. Click Add/Remove Windows Components. The Windows Components wizard appears.
  3. Internet Information Services is installed on Windows Whistler Server by default; therefore, the Internet Information Services (IIS) option should already be selected.
  4. Click the Details button. The Internet Information Services Add/Remove Components dialog box appears with a list of the optional subcomponents for IIS.
  5. Select the File Transfer Protocol (FTP) Service from the optional components list and click OK.
  6. Return to the Windows Components Wizard and click Next.
  7. FTP is installed.
