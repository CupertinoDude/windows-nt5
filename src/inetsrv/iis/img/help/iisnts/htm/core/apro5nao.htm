UseDigestSSP

Advanced digest authentication uses this metabase key as a switch between basic digest and advanced digest security support provider interface (SSPI) code. If the property is set to TRUE, the SSPI code for advanced digest is used. In all other cases (FALSE, empty or not set), IIS uses the basic digest SSPI code.

Data typeBoolean
Default valueFALSE
InheritanceInheritable
