Web Site Property Sheets

Each Web site on your computer, including the Default Web Site, can be configure to your specifications using the IIS snap-in.

To access the property sheets, right-click on the Web site listed in the Computer Management console and select Properties to view and configure properties listed below.

  • Directory Security Property Sheet
  • Documents Property Sheet
  • Custom Errors Property Sheet
  • Home Directory Property Sheet
  • HTTP Headers Property Sheet
  • Internet Information Services Property Sheet
  • ISAPI Filters Property Sheet
  • Operators Property Sheet
  • Performance Property Sheet
  • Web Site Property Sheet
