ODBC Logging Properties

Before you can begin ODBC logging from IIS, take these steps:

  1. Use ODBC-compliant database software (such as Microsoft Access or Microsoft SQL Server) to create a database on the computer being administered.
  2. Within the database, create a table containing the appropriate fields. Create a Data Source Name. See the Windows documentation for more information.
  3. Supply IIS with the Data Source Name and Table name by using this dialog box.
  4. If a user name and password are required for accessing the database, supply them as well.
