Advanced Configuration and Maintenance

This section covers enabling directory browsing, viewing site activity logs, and checking on site performance. You can use the settings to customize your site, or if you have security concerns and want to track who is accessing your site.

This section contains:

  • About Advanced Settings: Conceptual topic about the advanced settings available and how they can be used.
  • Changing the Default Home Page: Using a home page file other than Default.htm.
  • Enabling Directory Browsing: Using directory listings to provide navigation.
  • Logging Site Activity: Keeping a record of who visits your site.
  • Viewing Site Performance: Charting the number of connections made and refused.

This section does not contain information on:

  • Creating or changing publishing directories; see Adding New Publishing Directories.
  • Using scripts or applications in your Web pages; see Active Server Pages.
