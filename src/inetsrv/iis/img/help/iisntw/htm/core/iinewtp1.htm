Personal Web Manager Documentation

Welcome to the Personal Web Manager (PWM) documentation. PWM is a simplified Web site administration tool that you can use to quickly administer or monitor your Web site.

This section contains:

  • Creating and Customizing a Web Site: Enhancing your site with custom pages or an interactive database connection.
  • Adding New Publishing Directories: Changing the name or location of the home directory and adding additional content directories.
  • Advanced Configuration and Maintenance: Checking site performance, logging site activity, or changing the default document.

This section does not contain information on:

  • Internet Information Services setup or installation information; see Installing IIS.
  • Changing server settings by using Internet Information Services snap-in; see Administration.
