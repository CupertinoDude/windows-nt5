Backup Configuration

Use this dialog box to save configuration settings for all Web sites, FTP sites, virtual directories, directories, and files that you administer on your computer.

The Previous Backups list box lists all configuration backups that have been created, their version number, and the date that the backup was created.

To create a backup configuration, click the Backup button. This will save all current configuration settings on your computer.

Click the Delete button to delete a previous backup configuration.

Important    You cannot restore a backup configuration using Internet Services Manager (HTML).
