FTP Service

The FTP service is installed by default when IIS 5.1 is installed. A default FTP site is created at x:\Inetpub\Ftproot when the FTP service is installed, where x: represents the drive letter where IIS is installed. If IIS is installed and the FTP service is un-installed, you can reinstall the FTP service by following the procedure below.

To install FTP service
  1. In Control Panel, click Add/Remove Programs.
  2. Click Add/Remove Windows Components.
  3. From the list, select Internet Information Services (IIS), and click Details.
  4. Select the File Transfer Protocol (FTP) Server check box.
  5. Click OK (you may be prompted for the Windows XP CD or the network install pathname).
  6. Click Next.
  7. Click Finish.
