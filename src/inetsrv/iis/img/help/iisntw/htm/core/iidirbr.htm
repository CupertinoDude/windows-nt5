Enabling Directory Browsing

Directory browsing in an alternative way to provide navigation for visitors coming to your site. Instead of providing links on a home page to documents in the home directory, the default document is disabled and a list of all files in the directory is presented. This is particularly useful if you intend to share a large number of documents in their native format.

To enable directory browsing
  1. In Personal Web Manager, click the Advanced icon in the sidebar.
  2. In the Advanced Options panel, select the Enable Directory Browsing check box and clear the Enable Default Document check box.

Related Topics
