Internet Services Manager (HTML)

The HTML version of Internet Services Manager has a split-screen layout in your browser which consists of the menu view in the navigation pane of the browser and the tree view in the content pane of the browser.

The tree view shows all computers, Web sites, FTP sites, virtual directories, and directories that you can administer. Select any item in the tree view and it will appear highlighted in the browser. When an item is highlighted, you can then select a command from the menu view which will affect the highlighted tree view item.

The menu view lists the following commands:

New
Creates a new Web site or FTP site.

Delete
Deletes a Web site, FTP site, virtual directory, or directory.

Rename
Renames a Web site, FTP site, or virtual directory.

Properties
Displays properties for the selected tree view item.

Start
Starts the selected Web site or FTP site.

Stop
Stops the selected Web site or FTP site.

Pause
Pauses the selected Web site or FTP site.

Resume
Resumes the selected Web site or FTP site.

Permissions Wizard
Sets Web and FTP permissions, access permissions, and authentication schemes.

Master Properties
Sets the default values used by all current or new Web and FTP sites on a computer.

Backup Configuration
Saves all settings configured during an Internet Services Manager (HTML) session.

Caution   When using a non-secure connection, the only safe way to use Internet Services Manager (HTML) is through Secure Sockets Layer (SSL).
