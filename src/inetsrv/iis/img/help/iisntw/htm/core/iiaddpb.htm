Adding New Publishing Directories

As your site grows in complexity, or if you begin using scripts and applications in your Web pages, you may want to add additional directories and specify the type of access permitted to them. This section describes the process of expanding your Web site and defines related terminology.

This section contains:

  • About Publishing Directories: Definitions for home and virtual directories, and how they are used.
  • Changing Your Home Directory: Procedural help on changing the default home directory.
  • Adding Virtual Directories: How to add new and remove existing virtual directories, and what the permissions settings mean.

This section does not contain information on:

  • Creating content for your publishing directories; see Creating and Customizing a Web Site.
  • Creating or using scripts or applications on your site; see Active Server Pages.
