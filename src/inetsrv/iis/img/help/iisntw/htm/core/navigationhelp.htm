Using the IIS Help features

The Internet Information Services online documentation includes several features to help you find and use the information you need. You can navigate through the IIS Help using a mouse or using the keyboard.

Table of contents

  • In the navigation pane, you'll find a table of contents that lists all available features and topics.
  • When you select or click a top-level topic title, a page describing the contents of that section appears in the topic pane.
  • When you select or click the plus sign for the topic title, the table of contents expands so that you can see all the topics within that section.

Search

  • On the Search tab in the navigation pane, type a word or phrase in the Search for text box, and select the Search button.
  • Select your search type from the Select type of search text box. The default, Standard Search, searches for several variations of the word or phrase. You can choose Exact Phrase, Any Words, All Words, or Boolean Search to narrow the scope of your search.

Index

  • On the Index tab in the navigation pane, type a word in the text box to advance to the portion of the index that is closest to the word you typed.
  • Double-click or press ENTER on the selected entry to see that file or a list of topics that contain information related to the keyword.

Printing

To print one or more topics, select or click the Print button.

Synchronizing the table of contents with Index and Search results

After you select a topic from the index or search tools, you can see where that topic appears in the table of contents. When you view a topic from the Index or Search tabs, the topic appears in the right pane of the documentation window.

  • After viewing your topic from the Index or Search tab, select or click the Contents tab, and select or click the Synchronization button.

Navigating Through IIS Help Using the Keyboard

The TAB key serves as the navigational key.

  • Pressing TAB allows you to navigate from pane to pane, and to navigate the links within each pane, including the Content, Search, and Index tabs in the pane containing the table of contents.
  • Shift+TAB reverses the direction of movement.

Selecting a topic or item from the contents pane

  • Pressing ENTER on any selection results in its display. If the selection is a parent item in the table of contents, subordinate topics are expanded.
  • To navigate through the entire table of contents, use the UP and DOWN ARROW keys or the PAGE UP and PAGE DOWN keys.

Selecting a topic from the index pane

  • Select the index term you want to search on either by typing the index term in the text box or choosing it from the index terms presented.
  • Pressing ALT+D displays the topic or additional selections.

Selecting a topic from the search pane

  • Type the search term into the Search for text box.
  • Select the search type by pressing the TAB key to navigate to the list box and pressing the ALT+UP or DOWN ARROW key.
  • Use the TAB key to navigate to the Search button, and press ENTER to display the search results.
