Logging Site Activity

You can create a log file that lists everyone who has visited your site and which files they accessed while they were there. This is one way of checking who is using your site.

To enable site activity logging
  1. In Personal Web Manager, click the Advanced icon on the sidebar.
  2. In the Advanced Options panel, select the Save Web Site Activity Log check box.

Related Topics

  • For more information about where logs are saved, how they are named, and the log file format, see Activity Logs and Security in About Advanced Settings.
