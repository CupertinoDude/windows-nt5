Creating and Customizing a Web Site

IIS makes setting up your Web site quick and simple. This section walks you through the basics of creating your site. You also learn how to create an interactive form that stores information in a database.

This section includes:

  • About Web Sites: Conceptual information about what a Web site is and how it can be used.
  • Constructing an Interactive Form: Using cut-and-paste and search-and-replace techniques to customize an interactive response form.
  • Viewing a Database in a Browser: Accessing a database by using a sample script to display and edit the data in a Web browser.

This section does not include information on:

  • Changing the publishing directories; see Adding New Publishing Directories.
  • Tracking visitor activity or site performance; see Advanced Configuration and Maintenance.
