Using the Documentation

The Internet Information Services online documentation includes several features to help you find and use the information you need.

Navigation

  • In the navigation pane, you'll find a table of contents that lists all available features and topics.
  • When you click a top-level topic title, a page describing the contents of that section appears in the topic pane.
  • When you click the plus sign for the topic title, the table of contents expands so that you can see all the topics within that section.

Search

  • Click the Search tab in the navigation pane.
  • Type a word or phrase in the text box on the Search page.
  • The default, Standard Search, searches for several variations of the word or phrase. You can choose Exact Phrase, Any Words, All Words, or Boolean Search to narrow the scope of your search.

Index

  • To view the index, click the Index tab in the navigation pane.
  • Type a word in the text box to advance to the portion of the index that is closest to the word you typed.
  • Double-click an entry to see that file or a list of topics that contain information related to the keyword.

Printing

To print one or more topics, click the Print button.

Synchronizing the Table of Contents

To see where a topic appears in the table of contents, click the Synchronization button.
