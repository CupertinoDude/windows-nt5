The IIS Snap-in

The Internet Information Services snap-in provides server management options to control content and access to your Web or FTP sites. For example, if you are a developer testing a site before uploading it to a corporate intranet or the Internet, you can use this tool to test your settings exactly as they will be on the final server. The snap-in includes these features:

  • Additional server settings, such as settings to manage an FTP site, isolate applications, map MIME types, or designate additional script engines.
  • A wizard to create new virtual directories.
  • The ability to administer installations of Internet Information Services across a network.

To open the IIS Snap-In

Method One
  1. From the Start menu, select Settings and then Control Panel .
  2. Open Administrative Tools
  3. Open Internet Services Manager.
Method Two
  1. Right click on My Computer on your desktop
  2. Select Internet Information Services under Services and Applications

In addition you can create a shortcut to %SystemRoot%\System32\inetsrv\inetmgr.exe. Search for the keywords "add a shortcut" in Windows 2000 help from the Start menu for instructions about creating a shortcut.

Note   The Internet Information Services snap-in is an administration tool for IIS 5.1 that has been integrated with other administrative functions of Windows 2000. In previous releases this tool was called the Internet Service Manager.
