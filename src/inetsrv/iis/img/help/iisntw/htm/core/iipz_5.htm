Home Directory Property Sheet

You can change the location of the home directory to one of the following:

  • A directory located on the server you are configuring.
  • A share located on another computer.

When prompted, type the user name and password needed to access that computer.

FTP Site Directory

In the text box, type the path to the directory or the destination URL. The syntax must match the selected path type:

For a local directory, use the full path; for example, C:\Catalog\Shoes. You can also click the Browse button to select a local directory rather than typing the path.

For a network share, use a Universal Naming Convention (UNC) server and share name; for example, \\Webserver\Htmlfiles.

To select a local directory rather than typing the directory path in the Local Path text box, click Browse.

Read

To enable users to read or download files stored in a home directory or a virtual directory, select this option.

Write

To enable users to upload files to the enabled directory on your server, select this option. Select this permission only for directories that are intended to accept files from users.

Log access

To record visits to this directory in a log file, select this check box. Visits are recorded only if logging is enabled for this FTP site. Logging is enabled by default.
