Changing the Default Home Page

When you install IIS, a default home page is displayed. This is the file visitors see when they type the address of your site in their browser Address bar. If you have another file you want to use as your home page, and it does not have the name Default.htm, you can change the file name that the server looks for if a request does not specify a file name.

To change the default home page
  1. In Personal Web Manager, click the Advanced icon on the sidebar.
  2. In the Advanced Options panel, select the Enable Default Document check box and type the name of your file or files in the Default Documents box. If you use multiple file names, each name must be separated by a comma.

Related Topics

  • For more information on using multiple default documents, see The Role of Default Documents in About Advanced Settings.
