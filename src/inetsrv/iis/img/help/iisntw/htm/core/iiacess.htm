Accessibility For IIS

The following shortcut key procedures (or keyboard commands) are specific to IIS. To view the complete list of shortcut keys for Windows, look for "Start menu: keyboard commands" in the Windows Help index. For more specific information about navigation within the Microsoft Management Console (MMC), look for "MMC: accessibility" in the Windows Help index.

To access the IIS snap-in
  1. Click Start and then click Run. The Run dialog box appears.
  2. Type mmc %systemroot%\system32\inetsrv\iis.msc
  3. Press Enter. The Internet Information Services snap-in appears.

Note  If the IIS snap-in did not appear, check to make sure you typed in the command properly. There is a single space after mmc and before the percent sign.

You can also access IIS from the Computer Management Window. Accessing IIS in this way does not give you the range of administration options offered by the IIS snap-in, but it does offer quick access and limited management options for your Web sites.

  1. Click Start and then right-click My Computer. Click Manage from the drop-down list. The Computer Management window appears.
  2. In the left pane, expand the Services and Applications tree.
  3. Click Internet Information Services. The names and states of your Web sites appear in the right pane.

Note  Expand the Internet Information Services tree in the left pane and any subsequent Web site trees to see a list of directories and virtual directories for that Web page.

To access a particular property sheet
  1. In the IIS snap-in, use the arrow keys to select the site, virtual directory, or file you want to administer. In order to access files, you need to switch to the details pane. Use the shortcut key ALT to switch between the console tree and details pane in the IIS snap-in.
  2. Press the shortcut keys SHIFT+F10 to open the Action menu for the selected item.

    3. Note   To open the View menu, use the shortcut keys CTRL+A.

  3. Use either the UP or DOWN ARROW key to select Properties. Press ENTER.

    4. Note   Typing P will pause your site.

  4. Press the shortcut key TAB until the property sheet title is highlighted.
  5. Use either the RIGHT or LEFT ARROW key to select the correct property sheet on the same row.
  6. To switch between property sheet rows, use either the UP or DOWN arrow key.
  7. To work within that property sheet, press the shortcut key TAB. You can use the shortcut key TAB to select items or use the shortcut keys ALT+<underlined letter>.

    8. NOTE   To select or clear a check box, select it and press the SPACEBAR.

To navigate within the IIS online documentation
  1. If the Contents tab is not on top, press either the shortcut key TAB key or the shortcut keys SHIFT+TAB, until the Contents tab is highlighted.
  2. Bring the tab to the front by pressing ENTER.

    3. Note   You can switch between the Contents, Search, and Index tabs using this procedure.

  3. Press the shortcut key TAB key until an item in the table of contents is highlighted.
  4. Use either the UP or DOWN ARROW key to select the item.
    5. .
  5. To open a book, use the RIGHT ARROW key.
  6. To close a book, use the LEFT ARROW key.
  7. To open a document, select it and press ENTER.
