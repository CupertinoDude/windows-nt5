Internet Server Sample Filter: UpCase

"Where do you want to uppercase today?"
This sample works by monitoring requests for .htm files in a directory path that ends with /UC. It then strips the trailing /UC from the directory path and uppercases the body of the HTML text that goes by.

To install this filter, use the Internet Service Manager in the Start menu. 

Start\
  Programs\
    Windows NT 4.0 Option Pack\
      Microsoft Internet Information Server\
        Internet Service Manager

Right mouse click on the Web Site you wish to install the filter on. Choose Properties. Click on the ISAPI Filters tab, and add the filter.

Like any ISAPI Filter, this DLL is loaded when the web server is started and it stays loaded until the server is stopped.
