Microsoft SMTP Service Component Description

Microsoft SMTP service uses the SMTP Internet standard to transport and deliver messages based on specifications in request for comment (RFC) 821 and RFC 822. It also includes enhancements that build upon the basic delivery functions of the protocol. There are options available that give you control over message routing and delivery and that provide secure communications.

Component Configuration

There are no configuration requirements for this component.

For More Information

For more information on configuring the SMTP service, see Windows XP online Help. To view the SMTP Help for Windows XP Professional or Windows Whistler Server, use the following procedure:

  1. From the Start menu, choose Run.
  2. In the Open box, type %systemroot%\help\news.chm, and then choose OK.
