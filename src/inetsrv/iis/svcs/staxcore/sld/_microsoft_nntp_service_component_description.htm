Microsoft NNTP Service Component Description

Microsoft Network News Transfer Protocol (NNTP) service is used for both client/server and server-to-server communications. Microsoft NNTP service supports popular NNTP extensions and is fully compatible with other NNTP clients and servers.

For More Information

You can find information on configuring the NNTP service in the Windows XP online Help. Use the following procedure to view the NNTP Help on a Windows Whistler Server:

  1. From the Start menu, choose Run.
  2. In the Open box, type %systemroot%\help\news.chm, and then choose OK.
