SAPI Core Component Description

The Microsoft Speech application programming interface (SAPI) core component provides basic speech functionality. One component file, sapi.dll, is required for all speech functions and must be present. The other component file, sapisvr.exe, is unique to speech recognition and may not always be present after initial operating system (OS) installation.

Component Configuration

No direct configuration can be performed. Individual components, such as voices, may be configured or changed.

Special Notes

SAPI core files may not be tested directly. Because they are needed for speech operations, successful testing of related speech functions confirms their presence and current state.

To test the SAPI core component

  1. Open Speech in Control Panel.
  2. Select the Text-to-Speech tab.
