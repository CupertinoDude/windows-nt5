SAPI Common Component Description

The Microsoft Speech API (SAPI) common component represents basic functionality common to both text-to-speech and speech recognition. It is unlikely that any system other than speech needs to access these dynamic link libraries directly.

Component Configuration

There are no configuration requirements for this component.

Special Notes

SAPI common component files cannot be tested directly. Because these files are required for speech operations, the successful testing of related speech functions confirms their presence and current state.

To test text-to-speech

  1. Choose Start, point to Settings, choose Control Panel, and then double-click Speech.
  2. On the Text-to-Speech tab, choose Preview Voice. The text spoken uses the current voice and language.
