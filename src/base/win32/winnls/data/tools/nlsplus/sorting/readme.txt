These are the files and Perl scripts used to generate NLS+ data files.

Table generation
=================

Please run RunMe.cmd to generate the following file:
   * sortkey.nlp
   * sorttbls.nlp

After executing RunMe.cmd, please check in these files into Globalization\Tables.

File descriptions
=================

RunMe.cmd	Run this file to generate *.nlp files.
Unisort.txt	The Win32 sorting information source.
