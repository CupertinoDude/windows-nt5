Windows Uninstall

Setup is now removing Windows XP and is restoring your previous version of Windows. This process can take several minutes. When the uninstall is complete, your computer will automatically restart.

If you installed programs after you upgraded to Windows XP, you might need to reinstall them after this process is complete. If you created new documents in Windows XP, they will remain on your hard disk after your previous version of Windows is restored. Some of these documents might be moved to the Documents and Settings folder that Windows XP created on your computer.
