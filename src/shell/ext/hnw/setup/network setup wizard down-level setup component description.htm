Network Setup Wizard Down-Level Setup Component Description

This component consists of one executable file, called netsetup.exe. It installs the Network Setup Wizard, Universal Plug and Play (UPnP) and Internet Connection Sharing programs on Windows 98, Windows 98 SE, and Windows ME. This component uses IExpress as the base setup tool for unpackaging and installing the required binary files. It is not intended to run on Windows Embedded.

Component Configuration

There are no configuration requirements for this component.
