Server Print Component Description

This component supports local printing and printing to a server. It also supports Internet printing, printing to a printer queue exposed by the Line Printer Daemon (LPD) protocol, known as request for comment (RFC) 1179, and printing to a TCP/IP connection.

TCPMon supports the Line Printer Remote (LPR) and LPD protocols. TCPMon can use LPD to communicate with a printer if that printer supports LPD. LPRMon is only recommended for use with UNIX servers.

Component Configuration

There are no configuration requirements for this component.
