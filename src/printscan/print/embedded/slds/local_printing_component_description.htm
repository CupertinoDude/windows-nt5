Local Printing Component Description

This component supports printing over locally attached printers and printing without a server. This component includes the core spooler and router as well as the local spooler print provider. This component supports printers connected over the LPT 1284 port, the communications (COM) port, the File port, infrared (IR), universal serial bus (USB), 1394, Institute of Electrical and Electronics Engineers (IEEE) 1394.3, and using Dot4.

This component functions as the core printing service component. Ideally it should not contain any user interface (UI) components.

Component Configuration

There are no configuration requirements for this component.
