HP4LJRES

SUMMARY

This is a sample resource DLL particular to PCL 5\5e technology with support for Far East device fonts. It is 64-bit compliant.

BUILDING THE SAMPLE

To build the sample, type build -c. Once built, the DLL will produce one binary: hp4ljres.dll. There are no known bugs or issues to be documented.
