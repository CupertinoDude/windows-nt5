EP9RES

SUMMARY

This is a sample resource DLL particular to Epson 24 Pin technology with support for device fonts. It is 64-bit compliant and supports Canon CAPSL technology.

BUILDING THE SAMPLE

To build the sample, type build -c. Once built, the DLL will produce one binary: ep9res.dll. There are no known bugs or issues to be documented.
