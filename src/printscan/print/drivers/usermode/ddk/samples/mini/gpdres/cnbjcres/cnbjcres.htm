CNBJCRES

SUMMARY

This is a sample resource DLL particular to Canon Bubble Jet technology. It is 64-bit compliant.

BUILDING THE SAMPLE

To build the sample, type build -c. Once built, the DLL will produce one binary: cnbjcres.dll. There are no known bugs or issues to be documented.
