HPPJRES

SUMMARY

This is a sample resource DLL particular to Hewlett-Packard PaintJet technology with support for device fonts. It is 64-bit compliant and supports HP PaintJet technology.

BUILDING THE SAMPLE

To build the sample, type build -c. Once built, the DLL will produce one binary: hppjres.dll. There are no known bugs or issues to be documented.
