<FONT SELECTION COMMAND IN PCL XL MINIDRIVER'S UFM FILES>

The format of font name for PCL XL printer is 16 characters (bytes).

E.g.
Typeface           Size          BaseFontName+Enhance
                                 1234567890123456
------------------------------------------------------
Arial              16 bytes     "Arial           "
Arial Bold         16 bytes     "Arial         Bd"
Arial Bold Italic  16 bytes     "Arial       BdIt"
Arial Italic       16 bytes     "Arial         It"
