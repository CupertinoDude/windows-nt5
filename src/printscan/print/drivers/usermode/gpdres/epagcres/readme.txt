
Explanation to each subdirectory.

jpn: Japanese GPD compilation.

km: Kernel mode DLL compilation (currently not used)
um: Userl mode DLL compilation

ctt: Code translation tables.
pfm: Font resource files.

gpd: Source files for GPD files.

etc: Other misc files.  Also contains the old GPC file
     and readme.tx file for reference.

fe:  Obsolete
chp: Obsolete
