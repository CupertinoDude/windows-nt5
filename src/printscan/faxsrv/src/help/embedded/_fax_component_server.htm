Fax Component Description

The Fax component allows users to send and receive faxes, track and manage fax activity, and access archived faxes using a local fax device attached to your computer, such as a fax modem. The Fax component includes the following:

  • The Fax Client Console, which allows users to view and manage incoming and outgoing faxes, and access archived faxes.
  • The Fax Cover Page Editor, which allows users to create and edit cover pages to use when sending a fax.
  • The Fax Send Wizard, which allows users to send cover-page-only faxes, consisting of sender and recipient information, and a note.
  • The Fax Configuration Wizard, which allows users to configure fax devices on a local computer.
  • The Fax Outlook Extension, which provides a user interface that extends Outlook and allows the user to configure the parameters of a fax mail message before sending it from Outlook.

In addition, the Fax component includes Fax application programming interface (API) functions, which are defined as part of the Fax Software Development Kit (SDK). The Fax component also includes the Fax COM Object Model for Windows 2000 and an Extended COM Object Model for Windows XP.

Component Configuration

There are no configuration requirements for this component.
