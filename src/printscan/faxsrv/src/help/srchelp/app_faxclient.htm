Using Fax Client

With Fax Client, you can send and receive faxes, and centrally monitor and manage incoming and outgoing faxes on multiple fax printers. Fax Client allows you access to these services using either a local fax, or a server-based fax device. A local fax means that you do not need to be connected to a network in order to send and receive faxes. A server-based fax eliminates the need for a local fax device, reducing acquisition and maintenance costs.

Fax Client, part of Microsoft Fax, is not installed by default during Windows Whistler Setup.

To install Fax Client

  1. Click Start, point to Settings, click Control Panel, and then double-click Add/Remove Programs.
  2. Click Add/Remove Windows Components.
  3. In the Windows Components wizard, select the Microsoft Fax check box.

Start Fax Client.

Notes

  • To start Fax Client after Microsoft Fax has been installed, click Start, point to Programs, point to Microsoft Fax, and then click Fax Client.
  • For more information about using Fax Client, click Help in the Fax Client console.
