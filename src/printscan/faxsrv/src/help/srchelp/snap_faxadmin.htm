Using Fax Service Administration

With Fax Service Administration you can send and receive faxes, and centrally monitor and manage incoming and outgoing faxes on multiple fax printers. Fax Service Administration allows you access to these services using either a local fax, or a server-based fax device. A local fax means that you do not need to be connected to a network in order to send and receive faxes. A server-based fax eliminates the need for a local fax device, reducing acquisition and maintenance costs.

Fax Service Administration, part of Microsoft Fax, is not installed by default during Windows  Setup

To install Fax Service Administration

  1. Click Start, point to Settings, click Control Panel, and then double-click Add or Remove Programs.
  2. Click Add/Remove Windows Components.
  3. In the Windows Components wizard, select the Microsoft Fax check box.

Start Fax Service Administration.

Notes

  • To open Fax Service Administration, click Start, point to Programs, and then click Fax Service Administration
  • For information about using Fax Service Administration, click Help in the Fax Service Administration console.
