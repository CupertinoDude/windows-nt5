Instructions for using the Auto com test tool

 

The code in this directory is VB source code, and compiled DLL’s.

In order to use it, you should install AutoCOM (available in Microsoft Tool Box).

In order to run this test, type TEST.BAT at prompt.

 

Notice:

1.     After you install autocom, you must restart computer (although the installation don’t demand it)

2.     Open all project in VB, save them and re-create the DLL (File | Compile Fax?????.DLL in VB) before you start, otherwise you might get the error: “CLSID/PROID not registered error” when running.
