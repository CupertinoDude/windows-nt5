Additional help on the Internet

This topic lists some of the resources available on the Internet.

Microsoft Domain Migration and Restructuring

The Microsoft Domain Migration and Restructuring tools Web site contains up-to-date information and scenarios describing the various tools and methods used to migrate Windows NT networks to Windows 2000 networks and to restructure existing Windows 2000 networks.

For more information, or to download the most recent version of Active Directory Migration Tool, see the Microsoft Web site. (http://www.microsoft.com/)

Microsoft Personal Online Support

The Microsoft Personal Online Support Web site contains an extensive collection of articles, troubleshooting wizards, items you can download, and other useful product information, such as frequently asked questions (FAQs) and information from newsgroups.

For more information, or to begin your search, see the Microsoft Web site (http://www.microsoft.com/).

TechNet

TechNet is a service that contains resources for information technology (IT) professionals. This includes access to articles written by support professionals at Microsoft, career and industry information, discussion groups, technical feedback, and software. For more information about TechNet, see the Microsoft Web site (http://www.microsoft.com/).
