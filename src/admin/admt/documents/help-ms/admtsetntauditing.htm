To enable auditing on a Windows NT domain

  1. On a computer with administrative access to the Windows domain, click Start, point to Programs, point to Administrative Tools, and then click User Manager for Domains.
  2. On the Policies menu, click Audit.
  3. Click Audit These Events, and then next to User and Group Management, select the Success and Failure check boxes.

Note

  • You must be logged on as an administrator or a member of the Administrators group in order to complete this procedure.
