Password Options

Specifies how Active Directory Migration Tool assigns passwords to migrated user accounts. The options are defined as follows:

Complex passwords

Automatically generates a complex password for each migrated user account. Active Directory Migration Tool records the new passwords in the file specified in Location to store password file.

Same as user name

Sets the password for each copied user account to the first 14 characters of the user account name. Active Directory Migration Tool records the passwords in the file specified in Location to store password file.

Location to store password file

Specifies a file to which the assigned or generated passwords are written. You can type the full path to the file or use Browse to select a folder.
