Trust Information

Displays information about the trusts for the source and target domains. Options available in the Trust Migration Wizard ensure that the target domain has all the trusts that the source domain has.

To copy a trust for the target domain, select the desired trust, and then click Copy Trust. Active Directory Migration Tool prompts you for a user account and password that has administrative rights in the trusted domain. Specify the user account and password, and then click OK.

When you have finished selecting all the trusts you want to copy, click Next.
