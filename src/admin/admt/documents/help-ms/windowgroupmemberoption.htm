Group Member Options

Specifies how Active Directory Migration Tool handles members of the migrated groups. These options are defined as follows:

Generate complex passwords

Automatically generates a complex password for each migrated user account. Active Directory Migration Tool records the new passwords in the file specified in New password output file.

Set password to user name

Sets the password for each copied user account to the first 14 characters of the user account name. Active Directory Migration Tool records the passwords in the file specified in New password output file.

New password output file

Specifies a file to which the assigned or generated passwords are written. You can type the full path to the file or use Browse to select a folder.

Disable source accounts

Disables the user account in the source domain. This option prohibits anyone from using the source account to log on to the source domain.

Disable target accounts

Disables the user account in the target domain. This option prohibits anyone from using the new target account to log on to the target domain.

Leave both accounts active

Leaves the accounts in both the source and target domains active. When this option is selected, both accounts can be used to log on to either the source or target domains.

Days until source account expires

Sets the number of days after which the source account will no longer be available. This setting prohibits anyone from using the source account to log on to the domain after the specified number of days.

Translate roaming profiles

Copies roaming profiles from the source domain to the target domain for the selected user accounts. This process associates the roaming user profile with the new user account in the target domain.
