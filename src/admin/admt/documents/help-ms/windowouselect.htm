Organizational Unit Selection

Specifies the target organizational unit in the target domain.

Target OU

Specifies the target container, such as an organizational unit (OU), to which Active Directory Migration Tool will migrate selected user accounts, groups, or computer accounts. Specify a value in the following format:

container=name

Enter the full LDAP distinguished name of the target container. For example, to specify the organizational unit named TargetOU, in the testdomain.microsoft.com domain type:

LDAP://TESTDOMAIN/OU=TargetOU,DC=testdomain,DC=microsoft,DC=com

To select the desired container from the list of available containers, click Browse, and then follow the instructions on the Select Container page.
