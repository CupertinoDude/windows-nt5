User Account

Specifies the user account name and password that the Active Directory Migration Tool uses to collect information for the reports. The specified user account must have the necessary access permissions to collect the required information. The fields are defined as follows:

User name

Specifies the name of the user account the Active Directory Migration Tool uses to collect the information for the reports.

Password

Specifies the password for the user account specified in User account name.

Domain

Specifies the domain in which the user account is defined.
