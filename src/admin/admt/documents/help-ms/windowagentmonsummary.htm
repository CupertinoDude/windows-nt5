Summary tab

Displays the summary of the agents dispatched to the selected computers. The status options are defined as follows:

Computers started

Specifies the computers on which the agent has been successfully started.

Computers finished

Specifies the computers on which the agent has finished the specified actions.

Computers installed

Specifies the number of computers on which the agent has been successfully installed.

Computers running

Specifies the number of computers on which the agent is running.

Computers finished

Specifies the number of computers on which the agent has finished the specified actions.

Computers failed

Specifies the number of computers on which the agent was not able to be installed and run.

The following information provides a summary of the agent actions. The rows and columns are defined as follows:

Rows

Files

Specifies the number of files processed by the agent.

Directories

Specifies the number of directories processed by the agent.

Shares

Specifies the number of shares processed by the agent.

Group members

Specifies the number of members in local groups processed by the agent.

User rights

Specifies the number of entries for user rights processed by the agent.

Columns

Examined

Specifies the number of security descriptors that the agent searched for SIDs that match the SIDs of the selected accounts.

Changed

Specifies the number of security descriptors that the agent changed.

Unchanged

Specifies the number of security descriptors that the agent did not change.
