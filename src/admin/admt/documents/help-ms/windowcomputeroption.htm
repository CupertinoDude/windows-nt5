Computer Options

Specifies how Active Directory Migration Tool handles computers after it migrates the computer accounts to the target domain. The options are defined as follows:

Minutes before computers restart after wizard completion

Specifies how many minutes after the agent is dispatched and after the agent's task is completed that the computer on which the agent was run will be restarted.

Do not rename computers

Tries to name the migrated computer account the same name as the computer account in the source domain. If a naming conflict occurs, the options specified on the Naming Conflicts page control how the migrated computer account is named.

Rename with prefix

Adds the specified prefix to the name of each migrated computer account in the target domain. The original computer account with the same name in the target domain remains unchanged.

Rename with suffix

Adds the specified suffix to the name of each migrated computer account in the target domain. The original computer account with the same name in the target domain remains unchanged.
