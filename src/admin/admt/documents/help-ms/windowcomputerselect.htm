Computer Selection

Specifies which computers the Active Directory Migration Tool will migrate. To add one or more computers, click Add, and then follow the instructions on the Select Computers page. To remove one or more computers, select the desired computers, and then click Remove.

Note

  • Domain controllers cannot be migrated. To migrate a computer that is operating as a Windows 2000 domain controller, you must first demote that domain controller to a member or stand-alone server. Windows NT domain controllers cannot be migrated. You must either upgrade them to Windows 2000 Server or reinstall Windows NT.
