Monitoring Settings tab

These options are defined as follows:

Dispatch log

Specifies the location of the log file in which the agent records its actions. This option is only available after selecting Stop Monitoring.

Refresh interval

Specifies the amount of time, in seconds, the agent waits before sending the latest status information. This option is only available after selecting Stop Monitoring.

Start Monitoring

Enables agent monitoring.

Stop Monitoring

Disables agent monitoring and makes the Dispatch log and Refresh interval options available.
