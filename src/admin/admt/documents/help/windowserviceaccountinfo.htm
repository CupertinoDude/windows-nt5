Service Accounts Information Window

Allows you to view the service account information you have collected through the Service Account Migration wizard. You can select which Service Control Manager (SCM) entries you want to automatically update when the service account is migrated.

The Skip/Include button allows you to change the value in the Status column for the selected service. To update the SCM entry for a service when the service account is migrated, ensure the Status column is set to Include for that service.

During the user migration process, you can also specify whether you want any of the displayed services updated. If you want to skip all service accounts during the user migration process, click Migrate no service accounts.

If the tool was unable to update a service when it migrated the service account used by that service, the Service Account Migration wizard displays this window with the Update Failed status for that service. When that service is available, you can select the service and click Update SCM Now to update the Service Control Manager entry. If the SCM entry was successfully updated, the Status column is set to Updated.
