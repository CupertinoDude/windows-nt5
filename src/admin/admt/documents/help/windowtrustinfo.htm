Trust Information Window

Allows you to view information about the trusts for the source and target domains. The Trust Migration wizard allows you to ensure the target domain has all the trusts that the source domain has.

To copy a trust for the target domain, select the desired trust and click Copy Trust. The Active Directory Migration Tool prompts you for a user account and password that has administrative rights in the trusted domain. Specify the user account and password, and click OK.

When you have finished selecting all the trusts you want to copy, click Next.
