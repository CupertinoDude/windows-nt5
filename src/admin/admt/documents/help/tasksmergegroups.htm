To merge or map groups from one domain to another domain:

  1. Right-click Active Directory Migration Tool in the left pane of the main window.
  2. Click Group Mapping and Merging Wizard on the Active Directory Migration Tool menu.
  3. Follow the instructions until you have finished merging the desired groups. For more information about an option, click Help.
Notes:
  • If you specify a target group that does not already exist, the tool creates the target group in the specified organizational unit (OU). If you specify a target group that does already exist, the target group is moved to the specified OU.
  • If you have mapped a group to a different group in the target domain, and then you migrate that group from the source domain to the target domain, the mapping information is replaced. The group is then mapped to the migrated group in the target domain.
