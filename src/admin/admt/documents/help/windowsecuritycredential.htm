Security Translation Credentials Window

Allows you to specify the user account name and password the Active Directory Migration Tool uses to perform the security translation process. The specified user account must have the required access permissions to make the required changes. For example, the specified user account must have the following permissions:

  • Administrator rights in the source domain (member of Domain Admins)
  • Administrator rights in the target domain
  • Administrator rights on each computer you migrate
  • Administrator rights on each computer on which you translate security

If you want to translate Exchange security for Exchange mailboxes, distribution lists, custom recipients, organizations, sites, and containers, the user account you specify on this window must be a Permissions Admin in the Exchange site of the specified Exchange server.

The fields are defined as follows:

Domain
Specifies the domain in which the user account is defined.
User account name
Specifies the name of the user account the Active Directory Migration Tool uses to perform the security translation process.
Password
Specifies the password for the user account specified in the User account name text box.
