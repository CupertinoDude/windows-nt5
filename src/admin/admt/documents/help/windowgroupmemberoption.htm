Group Member Options Window

Allows you to specify how the Active Directory Migration Tool handles members of the migrated groups. If service accounts are members of the selected groups, the service accounts will be migrated based on the values you specified in the Service Account Migration Wizard.

These fields are defined as follows:

Generate complex passwords
Generates a complex password for each migrated user account. This option allows you to specify a file name in the New password output file field. The Active Directory Migration Tool records the new, generated passwords in the specified file.
Set password to user name
Sets the password for each copied user account to the first 14 characters of the user account name. This option allows you to specify a file in the New password output file field. The Active Directory Migration Tool records the passwords in the specified file. If you migrate service accounts, complex passwords are always generated for service accounts.
New password output file
Specifies a file to which the assigned or generated passwords are written.
Disable source accounts
Sets the Account disabled field on the user account in the source domain. This field prohibits anyone from using the source account to log on.
Disable target accounts
Sets the Account disabled field on the user account in the target domain. This field prohibits anyone from using the target account to log on.
Leave both accounts active
Directs the Active Directory Migration Tool to leave the Account disabled field unchecked for the account in both the source and target domains. This option allows both accounts to be used to log on.
Set source account to expire on
Sets the expiration date for the account in the source domain. This date prohibits anyone from using this account after the specified date to log on.
Translate roaming profiles
Copies roaming profiles from the source domain to the target domain for the selected user accounts. This process associates the roaming user profile with the new user account in the target domain.
