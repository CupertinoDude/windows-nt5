Group Options Window

Allows you to specify how the Active Directory Migration Tool handles the group migration. These fields are defined as follows:

Copy user rights
Copies the user rights assigned in the source domain to the target domain.
Copy group members
Migrates the members of the groups you selected to migrate. This option allows you to maintain the group membership information in the target domain. This option is not available in the Group Mapping and Merging wizard.
Remigrate previously migrated objects
Migrates the members of the groups you selected to migrate, even if those members were previously migrated. This options is available only if the Copy group members option is checked. This option is not available in the Group Mapping and Merging wizard.
Migrate group SIDs to target domain
Adds the SID of the migrated user accounts and groups in the source domain to the SID History of the new user accounts and groups in the target domain. This feature uses a secure connection to the source domain controller. For more information about SID History requirements, see Active Directory Migration Tool Requirements.
Do not rename groups
Tries to name the migrated group the same as the group in the source domain. If a naming conflict occurs, the options specified on the Naming Conflicts window control how the migrated group is named.
Rename with prefix
Adds the specified prefix to the name of each migrated group in the target domain. The original group with the same name in the target domain remains unchanged.
Rename with suffix
Adds the specified suffix to the name of each migrated group in the target domain. The original group with the same name in the target domain remains unchanged.
