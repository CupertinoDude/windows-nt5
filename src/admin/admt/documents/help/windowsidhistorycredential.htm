SID History Credentials Window

Allows you to specify the user account name and password the Active Directory Migration Tool uses to update the SID History for migrated accounts in the target domain. The specified user account must have Administrator rights in the source and target domains.

The fields are defined as follows:

Domain
Specifies the domain in which the user account is defined.
User account name
Specifies the name of the user account the Active Directory Migration Tool uses to update the SID History for migrated accounts.
Password
Specifies the password for the user account specified in the User account name text box.
