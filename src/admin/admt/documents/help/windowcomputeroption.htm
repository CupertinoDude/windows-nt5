Computer Options Window

Allows you to specify how the Active Directory Migration Tool handles computers after it migrates the computer accounts to the target domain. The fields are defined as follows:

Restart computers in ___ minutes
Automatically restarts computers migrated to the target domain to complete the migration process.
Do not rename computers
Tries to name the migrated computer account the same as the computer account in the source domain. If a naming conflict occurs, the options specified on the Naming Conflicts window control how the migrated computer account is named.
Rename with prefix
Adds the specified prefix to the name of each migrated computer account in the target domain. The original computer account with the same name in the target domain remains unchanged.
Rename with suffix
Adds the specified suffix to the name of each migrated computer account in the target domain. The original computer account with the same name in the target domain remains unchanged.
