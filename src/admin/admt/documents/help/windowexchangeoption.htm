Exchange Options Window

Allows you to specify the Exchange objects included in the security translation process. The fields are defined as follows:

Translate Exchange directory containers
Directs the Active Directory Migration Tool to translate the security on all the Exchange directory containers.
Translate containers
Specifies the names of the Exchange containers on the selected Exchange server that contains the objects you want to translate. The Active Directory Migration Tool translates the objects in the specified containers.
MAPI profile
Specifies the name of the MAPI mail profile to use for the translation.
Translate mailboxes
Specifies the names of the Exchange mailboxes on the selected Exchange server that you want to translate. The Active Directory Migration Tool translates the specified mailboxes.
