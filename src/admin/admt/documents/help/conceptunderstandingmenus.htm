Menus

The following menus are available in the Active Directory Migration Tool:
Console

Allows you to create new consoles, open saved consoles, and save changes you have made to the console. For example, you can add management components, such as snap-ins, to a console. The Console menu also allows you to set console options, such as opening console files in Author mode.

Window

Allows you to create new windows in the console, arrange existing windows, and switch between open windows.

Help

Provides links to the Help file and information on the Web. For more information about using the MMC, see the MMC Help.

Action

Displays commands that correspond to the item selected in the console window. These commands allow you to perform functions such as refreshing the view, selecting the domain, and opening a new window. You can also right click an item in the console window to display its context menu, which displays the same commands as the Action menu.

View

Allows you to specify how information is displayed in the console window. You can show or hide items, such as the console tree, description and status bars, and toolbars. The commands in the View menu correspond to the item selected in the console window.
