Windows® Management Instrumentation SNMP PROVIDER README


Microsoft WMI SNMP Provider

May 1999

 

In order to keep the base size of the WMI SNMP provider distribution as small as possible, a number of nonessential files have been removed and relocated to the WMI Software Development Kit (SDK).  Specifically, the IETF standard SNMP MIBs (e.g., RFC-1213) and precompiled WMI MOF and SMIR MOF versions of these MIBS can now be found in the WMI SDK. 

 

The WMI SDK may be downloaded from http://msdn.microsoft.com/developer/sdk/wmisdk/.
