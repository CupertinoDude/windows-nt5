Valid Flags for IWbemServices Methods

Rev 1.03, raymcc, 16-Jul-99 

Documentation of lFlags parameters in IWbemServices methods prior to 13-Jul-99 had a number of errors.  This document lists the correct set of flags as evidenced by the actual sources.

Note that WBEM_FLAG_SEND_STATUS is not supported except in Async APIs.   The previous documentation actually incorrectly listed WBEM_FLAG_SEND_STATUS as valid for Sync calls.
