Browsing AppManager jobs, events, and data

Each time a Knowledge Script runs, it can generate new events or collect new data. To browse events and data:

  1. Using a WMI browser, connect to the NetIQ namespace (<root>\netiq). Then:

    • To list currently running jobs, select the NetIQ_LRJob class and enumerate its instances.

    • To list events, select the NetIQ_LREvent class and enumerate its instances.

    • To list data points, select the NetIQ_LRGraph class and enumerate its instances.
