Thank you for trying the NetIQ AppManager WBEM Agent

The NetIQ® AppManager® Suite provides a comprehensive solution for managing and monitoring the performance and availability of distributed Windows 2000 and Windows NT systems, BackOffice applications such Exchange and SQL Server, and other server applications such as Lotus Domino. From a single, central console, network administrators can monitor the performance and health of their Windows 2000 and Windows NT systems and configure automated responses to network and application events, proactively addressing potential problems before they have a chance to affect network performance and availability. AppManager also provides extensive data-collection capabilities so you can analyze and report on the state of your network and application services.

AppManager provides an advanced solution for centrally managing virtually every component of your Windows 2000-based environment, as well as Windows NT Versions 4 and 3.51. NetIQs comprehensive and adaptable approach to Windows 2000/NT management has made it a trusted partner of Global 1000 accounts such as AT&T, EDS, General Electric, Glaxo Wellcome, MCI, Microsoft, NationsBank, Salomon Smith Barney, Shell Oil Company, The Southern Company, and Wells Fargo. NetIQs technological innovation and strong customer acceptance has also led to numerous industry awards.

In particular, Microsoft's IT Group has selected the NetIQ AppManager Suite to monitor the performance and availability of its global Windows NT Server deployment. Microsoft will use the NetIQ AppManager Suite to help proactively manage its worldwide Windows NT Server infrastructure, as well as centrally monitor business-critical server applications that run on Windows NT, such as Microsoft Exchange Server, Internet Information Server (IIS) and Microsoft SQL ServerTM.

This CD-ROM includes an introduction to NetIQ AppManager's support for Microsoft's Web-Based Enterprise Management (WBEM). The NetIQ AppManager WBEM Agent is a member of the NetIQ AppManager family of products; it is designed to work with the Web-Based Enterprise Management technology. For the version on this CD, we have selected some of the most useful and productive monitoring features for NT servers and workstations. These monitoring features come preconfigured with the most common default settings, and allow you to set some additional options. As you work with the AppManager WBEM Agent, however, please keep in mind that you are seeing only a small subset of the capabilities of the full AppManager Suite.
