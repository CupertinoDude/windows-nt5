Learning more about the AppManager Suite

AppManager offers additional management features for Windows 2000, Windows NT, BackOffice applications (including Microsoft Exchange Server, SQL Server, and Internet Information Server), and applications such as Lotus Domino.

Our products deliver not only unique depth of monitoring for your entire Windows 2000/NT environment but can also be easily adapted and tailored to meet your business needs using industry-standard technologies such as VBA scripting. Built on a highly scalable architecture that fully leverages Microsoft technology, the NetIQ AppManager Suite is the perfect complement to Microsoft's systems management offerings.

NetIQ's comprehensive product line provides an easy-to-use central console. From this console, you can configure AppManager to optimize performance, ensure availability through automated problem detection and correction, and lower the support costs associated with monitoring Windows NT systems and applications. The NetIQ AppManager Suite consists of fully integrated product modules, or "AppManagers," each of which manages one of the following products:
 

On the Web

Microsoft Windows NT (Workstation, Server and Enterprise Edition)

Microsoft Exchange Server

Microsoft SQL Server

Lotus Domino/Notes

Compaq Insight Manager

Microsoft Systems Management Server

Microsoft Internet Information Server

Microsoft Cluster Server

Microsoft Transaction Server

Microsoft Message Queue Server

Microsoft Proxy Server

Microsoft Commercial Internet System News Server
