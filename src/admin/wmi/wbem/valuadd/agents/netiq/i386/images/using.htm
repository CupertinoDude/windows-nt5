Using the AppManager WBEM Agent

To connect to a computer managed by the AppManager WBEM Agent, you need a WBEM-enabled browser, such as the WBEM Object Browser that is included in the WBEM Software Development Kit.   The easiest way to use the AppManager WBEM Agent is to use a WBEM console configured for the AppManager WBEM Agent.
