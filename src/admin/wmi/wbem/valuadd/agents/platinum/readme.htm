The purpose of this document is to provide you with information on the integration of WMI and WBEM with the ProVision suite of products from PLATINUM technology.
 
PLATINUM technology is committed to supporting the Microsoft Windows Management Instrumentation (WMI) standard. PLATINUM is active in the Web-Based Enterprise Management (WBEM) initiative through its work on the Desktop Management Task Force that is defining the WBEM standard and the CIM data model.
 

PLATINUM ProVision is a suite of highly integrated tools that address the complexities of managing your IT environment. ProVision's management tools enable users to identify, respond to, and resolve problems across multiple disciplines and multiple environments, including UNIX, Windows NT and heterogeneous relational databases.

The ProVision WMI Agent uses WMI technology derived from the WBEM initiative. The WMI Agent enables any ProVision tool to both consume and provide management data from and to WMI. Such capabilities are useful for managing WMI because ProVision features a sophisticated event management service with features like publish and subscribe messaging, guaranteed delivery, and event correlation.

The WMI Agent for ProVision can also provide better data integration for management applications and new ways to deliver custom reports and views into Windows or browser-based consoles like the PLATINUM Director.

For example, AutoSys Job Management for ProVision can consume and provide information about scheduled jobs running in the Windows environment and can take appropriate actions using the ProVision Event Management Service. ProVision Performance Management tools can collect system and application performance data from the entire enterprise, provide that information to WMI consumers, and exploit WMI information for analysis and reporting within ProVision.

Technical specifications for the WMI Agent for ProVision are available. For more information, download the readme.txt and the PLATINUM WMI white paper from our web site, www.platinum.com.
