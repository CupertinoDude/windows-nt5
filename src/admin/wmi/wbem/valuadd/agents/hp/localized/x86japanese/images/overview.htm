ManageX: 
集中的なポリシー ベースの NT 管理

HP OpenView ManageX は、Windows NT(R) システム管理者の最も重要なニーズに焦点を合わせた強力で使やすい管理ソリューションです。このデモ バージョンには、NT イベント ログのイベントに基づいて通知を送信する 2 つのポリシーが含まれていますが、ManageX のフルバージョンには、特に年中無休で 1 日 24 時間稼働しているような環境でのミッション クリティカルな Windows NT(R) システムや BackOffice[tm] アプリケーションの可用性とパフォーマンスを保証する、200 を超えるポリシーが用意されています。さらに、ManageX パフォーマンス モニタによって、パフォーマンスの特性がリアルタイムで診断されます。

学習的な管理

ManageX は、問題が発生した場合に通知を送信するだけでなく、問題が実際に起こる前にそれを発見する学習的な監視とレポートによって、NT への投資が最も有効に利用されるよう設計されています。フルバージョンでは、スマート レポータを使って、容量計画、傾向分析、トラブルシューティングなどに使用するシステムおよびアプリケーションのパフォーマンス データをログに記録できます。この情報には、考えられる原因と解決策が記述された、パッケージ化されているレポートからアクセスできます。

ManageX では、アプリケーション管理とイベント管理によって管理タスクを自動化することもできます。組み込まれている相関エンジンが例外ベースの管理と、自動化された修正アクションに対するカスタマイズ可能なトリガを提供します。

最先端の Windows NT(R) 管理ソリューション

HP OpenView ManageX は  Windows NT(R) と 100% 互換性があり、DCOM (分散コンポーネント オブジェクト)、WMI、ActiveX などの、Microsoft の最先端テクノロジーを基盤としています。ManageX は、MMC (マイクロソフト マネージメント コンソール) 標準に準拠する業界初のソリューションです。 Windows NT(R) に依存した環境では、ManageX が今すぐ管理ソリューションを提供します。
