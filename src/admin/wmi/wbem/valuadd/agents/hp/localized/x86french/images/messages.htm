Lecteur de messages

Le Lecteur de messages de ManageX centralise les messages générés par les stratégies de ManageX, ce qui permet de repérer rapidement une erreur.

ATTENTION :
Vous devez cliquer au moins une fois sur le Lecteur de messages à chaque fois que vous ouvrez une session avec ManageX de manière à activer son système de mise à jour dynamique. Vous ne pourrez pas recevoir de messages avant de l'avoir fait.

Affichage des messages

Le format tableau se présente sous forme de colonnes mettant continuellement à jour les informations que le Lecteur de messages génère pour chaque message. Ces informations comprennent le type du message, l'heure de réception, la source (application/service ainsi que l'ordinateur l'ayant émis), une ID d'événement et l'état de l'accusé de réception. Vous pouvez trier ces messages en cliquant sur tout titre du tableau.

Pour plus de détails sur un message, ouvrez sa page Propriétés à l'aide de son menu de raccourci.

Filtrage des messages

Lorsque ManageX est installé sur d'autres machines faisant partie de votre réseau, le Lecteur de messages affiche tous les messages générés par les stratégies installées et par ces autres machines. (Dans la version complète du produit, le Lecteur de messages centralise l'affichage des messages générés par les machines sur réseau sur lesquels ManageX a été installé. Ces messages ont trait aux activités internes de ManageX, aux informations du système, aux audits des succès et des échecs, aux avertissements et aux erreurs. Ils peuvent exiger l'intervention de l'administrateur ou lancer une procédure automatique.)

Vous pouvez éviter d'afficher les messages en provenance des autres machines en ne visualisant que les messages sélectionnés.

Pour visualiser les messages sélectionnés :

  1. Dans l'arborescence de console de MMC (volet de gauche), double-cliquez sur le nœud OpenView ManageX pour l'ouvrir puis cliquez une fois pour sélectionner l'extension Lecteur de messages.
  2. Passez à la fenêtre indépendante Sélecteur de périphériques.

    Si le Sélecteur de périphériques n'apparaît pas, cliquez à l'aide du bouton droit de la souris sur le nœud OpenView ManageX et sélectionnez Sélecteur de périphériques dans le menu de raccourci OU utilisez le bouton Sélecteur de périphériques dans la barre d'outils,

    puis vérifiez et cliquez sur l'onglet Périphériques. Sélectionnez une machine, puis cliquez sur Appliquer.

  3. A l'aide du bouton droit de la souris, cliquez sur Lecteur de messages et sélectionnez Affichage > Messages sélectionnés dans le menu de raccourci.
