Anzeigen von Meldungsdetails

Während im (rechten) Ergebnisfenster des Meldungslesers eine fortlaufend aktualisierte Tabelle von Meldungen zu sehen ist, können Sie auch ein Dialogfeld mit zusätzlichen Details zu jeder beliebigen Meldung aufrufen.

ACHTUNG:
Sie müssen in jeder ManageX-Sitzung mindestens einmal auf den Meldungsleser klicken, um sein sich dynamisch aktualisierendes System zu aktivieren. Erst dann erhalten Sie Meldungen.

Vorgehensweise:

  1. Suchen Sie in der tabellenartig aufgebauten Meldungsanzeige die Meldung, zu der Sie weitere Informationen sehen möchten.
  2. Klicken Sie mit der rechten Maustaste auf die gewünschte Meldung, und klicken Sie im Kontextmenü auf Eigenschaften.
    ODER
    Doppelklicken Sie auf die gewünschte Meldung.
  3. Wenn das Register Meldungsdetails angezeigt wird, können Sie sich die verfügbaren Daten ansehen, die auf einer oder zwei Seiten dargeboten werden.
    • Meldungsdetails zeigt die in der Meldung enthaltenen Informationen an, inkl. einer vollständigen Beschreibung.
    • Zusätzliche Info bietet Details zum Meldungs-Routing.
  4. Wenn Sie sich die Informationen angesehen haben, klicken Sie auf OK oder auf Abbrechen, um das Dialogfeld zu schließen.
