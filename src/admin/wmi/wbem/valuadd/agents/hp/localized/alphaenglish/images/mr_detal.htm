Viewing message details

While the Message Reader's results (right) pane displays a continuously updating table of messages, you can also access a dialog displaying additional details about any particular message.

CAUTION:
You must click on the Message Reader at least once in each ManageX session to activate its dynamically updating system. You will receive no messages until you have done this.

To view message details:

  1. In the tabular message display, locate the message about which you want more information.
  2. Right-click the desired message and choose Properties from the shortcut menu
    OR
    Double-click the desired message.
  3. When the Message Details tab displays, review the available information, available on one or two tabbed pages.
    • Message Detail displays information contained within the message, including a full description.
    • Additional Information provides information message routing details.
  4. When you finish reviewing the information, click OK or Cancel to close the dialog.
