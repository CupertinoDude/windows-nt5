Controlling message display properties

The Message Reader's results (right) pane displays a continuously updating table of messages received from other machines within your domain and subnet. You can use the extension's Properties command to change various aspects of the display's format and content.

CAUTION:
You must click on the Message Reader at least once in each ManageX session to activate its dynamically updating system. You will receive no messages until you have done this.

To control message display properties:

  1. In MMC's console tree (left pane), double-click the OpenView ManageX node to open it and then click once to select the Message Reader extension.
  2. Right-click the Message Reader node and select Properties from the shortcut menu.
  3. When the Properties dialog displays, make your selections. You can click Apply at any point to make your changes without closing the dialog.
    • In the Message History section, use the spin boxes to determine the number of messages that will be retained in the Message Reader tabular display list. (By default, the Message Reader retains the last 250 messages in a scrollable display list.)
    • In the Date section, select or clear the Show Date checkbox to determine whether or not the date column is displayed. You can also use this section's radio buttons to change the format of the date display between Long Date Format (eg., Monday, May 11, 1998) and Short Date Format (eg., 5/11/98). (By default, the date displays in short format.)
    • In the Time section, select or clear the Show Time checkbox to determine whether or not the time column is displayed. You can also use this section's radio buttons to change the format of the time display from the default choice of Local Time Format (that is, the time format appropriate to the system default locale set in your Windows Regional Settings) or 24 Hour Time Format (23:28:00).

      Note: The 24 Hour Time Format may give you more efficient sorting of messages by time.
    • The Configuration File section is not active in this version of ManageX
  4. When you finish making your selections, click OK to apply the changes or Cancel to revert to the previous display properties.
