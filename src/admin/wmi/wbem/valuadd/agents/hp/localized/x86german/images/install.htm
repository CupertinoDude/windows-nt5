Installation von ManageX

Für ManageX gibt es ein Standard-Installationsverfahren, das über die Datei setup.exe aufgerufen wird.

Installieren von ManageX

  1. Öffnen Sie die Datei setup.exe.
  2. Das Installationsverfahren wird automatisch gestartet. Die Installation gestaltet sich am einfachsten, wenn Sie die auf den Bildschirmen vorgeschlagenen Standardeinstellungen wie folgt übernehmen:
    Bildschirm Standardeingabe
    Willkommen Weiter
    Software- Lizenzvertrag Ja
    Zielpfad wählen Weiter
    (Standardeinstellung ist C:\Programmdateien\HP OpenView ManageX)
    Programmordner auswählen Weiter (Standardeinstellung ist HP OpenView ManageX)
    Kopiervorgang starten Weiter
  3. Am Ende des Installationsverfahrens wird ein Web-Browser mit einer ManageX-Informationsdatei geladen. Überprüfen Sie die Datei, und schließen Sie den Browser.
  4. Klicken Sie in dem nun eingeblendeten Dialogfeld Setup abgeschlossen auf Beenden, um es zu schließen und wieder den Desktop anzuzeigen.
