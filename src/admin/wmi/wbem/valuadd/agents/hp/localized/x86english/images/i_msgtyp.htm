Understanding message type icons

ManageX uses the following icons to identify each message by type. Note that each message type is assigned a priority level from 6 (high importance) to 1 (low importance).

Icon Type Priority
Error 6
Audit Failure 5
Warning 4
Audit Success 3
Information 2
ManageX Internal 1

These icons appear in the Message Reader's results (right) pane, where each message displays an icon identifying its type.

Caution:
You must click on the Message Reader at least once in each ManageX session to activate its dynamically updating system. You will receive no messages until you have done this.
