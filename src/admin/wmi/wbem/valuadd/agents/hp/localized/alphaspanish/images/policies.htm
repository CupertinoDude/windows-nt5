Directivas

Esta versión de ManageX incluye dos directivas que vigilan el registro de sucesos de NT y envían notificaciones al Lector de mensajes de ManageX:

  • NtCriticalEvents.mxw, que queda activada durante la instalación, envía mensajes sólo cuando se produzcan sucesos esenciales.
  • NtAllEvents.mxw, que queda desactivada durante la instalación, envía mensajes para todos los sucesos del registro de sucesos de NT.

Cambios en el estado de las directivas

ManageX incluye una utilidad de línea de comandos, MxPolicyMgr, que permite cambiar el estado predeterminado de las dos directivas.

Activación y desactivación de las directivas

  1. En la línea de comandos, cambie al directorio de instalación de ManageX. Si aceptó la instalación predeterminada, se tratará del directorio C:\Archivos de programa\HP OpenView ManageX.
  2. Para activar las directivas, escriba las dos líneas siguientes, pulsando Intro después de cada línea (si desea activar sólo una directiva, escriba únicamente la línea con el nombre de directiva correspondiente).

    MxPolicyMgr -enable Policies\NTCriticalEvents.mxw
    MxPolicyMgr -enable Policies\NTAllEvents.mxw
  3. Para desactivar las directivas, escriba las dos líneas siguientes, pulsando Intro después de cada línea (si desea desactivar sólo una directiva, escriba únicamente la línea con el nombre de directiva correspondiente).

    MxPolicyMgr -disable Policies\NTCriticalEvents.mxw
    MxPolicyMgr -disable Policies\NTAllEvents.mxw
  4. Vuelva a ManageX en Microsoft Management Console™ y haga clic en el Lector de mensajes para activarlo. A partir de este momento, aparecerán mensajes cada vez que se produzcan los correspondientes sucesos del registro de sucesos de NT.
