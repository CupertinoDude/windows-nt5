Message Reader

The ManageX Message Reader consolidates messages generated by your ManageX policies, allowing you to quickly see when an error condition occurs.

CAUTION:
You must click on the Message Reader at least once in each ManageX session to activate its dynamically updating system. You will receive no messages until you have done this.

Message Display

The default tabular format presents columns of continuously updating Message Reader information about each message, including the message type, time received, the source (application/service source as well as sending computer), an event ID, and acknowledgement status. You can sort these messages by clicking on any of the table headings.

Additional details about each message are available on a Properties page accessible from the message's shortcut menu.

Message Filtering

If other machines in your network domain have ManageX installed, your Message Reader displays all messages generated by your installed policies and by these other machines. (In the full product, the Message Reader provides a central location for the display of messages generated by all network machines on which ManageX has been installed. These messages may signal internal ManageX activity, system information, audit successes, audit failures, warnings, or errors, and may require administrator attention or initiate automated action.)

You can filter out messages from other machines by viewing only selected messages.

Viewing selected messages

  1. In MMC's console tree (left pane), double-click the OpenView ManageX node to open it and then click once to select the Message Reader extension.
  2. Go to the separate Device Selector window,

    If you do not see the Device Selector, right-click on the OpenView ManageX node and select Device Selector from the shortcut menu OR use the Device Selector button on the toolbar.

    and check and click on the Devices tab. Select your machine, then click Apply.
  3. Right-click the Message Reader and select View > Selected Messages from the shortcut menu.
