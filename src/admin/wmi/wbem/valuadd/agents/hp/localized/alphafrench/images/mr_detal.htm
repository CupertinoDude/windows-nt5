Visualisation des détails d'un message

Lorsque le volet de résultats du Lecteur de messages (à droite) affiche un tableau de messages continuellement mis à jour, vous pouvez également ouvrir une boîte de dialogue pour afficher des détails supplémentaires sur n'importe quel message.

ATTENTION :
Vous devez cliquer au moins une fois sur le Lecteur de messages à chaque fois que vous ouvrez une session avec ManageX de manière à activer son système de mise à jour dynamique. Vous ne pourrez pas recevoir de messages avant de l'avoir fait.

Pour visualiser les détails d'un message :

  1. Alors que les messages affichés sous forme de tableau, repérez le message au sujet duquel afficher des informations supplémentaires.
  2. A l'aide du bouton droit de la souris, cliquez sur le message voulu et choisissez Propriétés dans le menu de raccourci
    OU
    double-cliquez sur le message voulu.
  3. Lorsque l'onglet Détails du message s'affiche, prenez connaissance des informations proposées sur une ou deux pages de l'onglet.
    • Détails du message affiche les informations contenues dans le message, y compris une description complète.
    • Informations complémentaires fournit des détails sur le routage du message.
  4. Après avoir pris connaissance des informations, cliquez sur OK ou sur Annuler pour refermer la boîte de dialogue.
