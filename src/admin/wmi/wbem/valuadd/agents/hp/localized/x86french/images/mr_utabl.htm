Tableau d'affichage du Lecteur de messages

L'affichage par défaut sous forme de tableau du Lecteur de messages présente des informations sur tous les messages reçus en provenance des machines présentes sur votre domaine local et votre sous-réseau sur lesquelles ManageX a été installé. Chaque colonne de l'affichage présente des informations comme décrit ci-dessous.

ATTENTION :
Vous devez cliquer au moins une fois sur le Lecteur de messages à chaque fois que vous ouvrez une session avec ManageX de manière à activer son système de mise à jour dynamique. Vous ne pourrez pas recevoir de messages avant de l'avoir fait.

L'affichage sous forme de tableau devrait ressembler à l'image ci-dessous dans le volet de résultats de la console MMC (à droite) :

Description des colonnes :

  • Icône/Type — Indique le type de message reçu (Information, Avertissement, Erreur, etc.)
  • Heure — Date et heure locale de réception du message. Vous pouvez choisir leur format et leur affichage dans Propriétés.
  • Source — Source ou origine message. Bien que le nom de l'application d'origine soit souvent utilisé, cette valeur peut être définie par l'utilisateur et peut être un nom de sous-système (par exemple, SQL Server) ou tout autre nom descriptif.
  • Catégorie — Type d'événement ayant généré le message, par exemple START, STOP ou INSTALL.
  • ID d'événement — Numéro d'identification de l'événement.

    REMARQUE : Dans la version complète de ManageX, les IDs d'événement générées par les stratégies de ManageX sont listées avec recherche de texte dans l'aide en ligne.

  • Ordinateur — Nom de la machine ayant généré le message.
  • Réception confirmée — Indication par Oui/Non de la confirmation de réception d'un message. Un message peut être marqué lu et traité ou à réception confirmée, à l'aide de son menu de raccourci.
