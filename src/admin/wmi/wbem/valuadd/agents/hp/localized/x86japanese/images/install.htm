ManageX のインストール

ManageX では、setup.exe ファイルからアクセスする標準のインストール手順を使用します。

 ManageX のインストール

  1.  setup.exe を開きます。
  2. インストール シールドのインストール プロセスが自動的に起動します。画面に表示されるデフォルト (下表) を受け入れると、最も簡単にインストールできます。
    3. 画面 デフォルトの応答
    ようこそ 次へ
    ソフトウェア ライセンス契約書 はい
    インストール先の選択 次へ 
    (デフォルトは C:\Program Files\HP OpenView ManageX)
    プログラム フォルダの選択 次へ (デフォルトは HP OpenView ManageX)
    ファイルのコピー開始 次へ

  3. インストール過程の最後で、Web ブラウザが起動して、ManageX 情報ファイルが読み込まれます。ファイルを読んでから、ブラウザを閉じます。
  4. [セットアップの完了] ダイアログ ボックスが表示されたら、[完了] をクリックしてダイアログ ボックスを閉じ、デスクトップに戻ります。
