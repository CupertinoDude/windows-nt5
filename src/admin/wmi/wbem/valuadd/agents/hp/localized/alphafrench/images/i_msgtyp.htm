Icônes de type de message

ManageX utilise des icônes pour indiquer le type des messages. Ces icônes sont décrites ci-dessous. Vous remarquerez qu'à chaque type de message correspond un niveau de priorité allant de 6 (très important) à 1 (peu important).

Icône Type Priorité
Erreur 6
Audit des échecs 5
Avertissement 4
Audit des succès 3
Informations 2
Interne à ManageX 1

Ces icônes sont visibles dans le volet de résultats du Lecteur de messages (à droite). Une icône est accolée à chaque message pour en indiquer le type.

ATTENTION :
Vous devez cliquer au moins une fois sur le Lecteur de messages à chaque fois que vous ouvrez une session avec ManageX de manière à activer son système de mise à jour dynamique. Vous ne pourrez pas recevoir de messages avant de l'avoir fait.
