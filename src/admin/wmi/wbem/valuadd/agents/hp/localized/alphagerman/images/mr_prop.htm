Steuerung der Meldungsanzeige

Im (rechten) Ergebnisfenster des Meldungslesers wird eine fortlaufend aktualisierte Tabelle von Meldungen angezeigt, die von anderen Computern in der Domäne oder im Teilnetz empfangen wurden. Mit dem Befehl Eigenschaften der Erweiterung können Sie verschiedene Aspekte hinsichtlich Format und Inhalt der Anzeige ändern.

ACHTUNG:
Sie müssen in jeder ManageX-Sitzung mindestens einmal auf den Meldungsleser klicken, um sein sich dynamisch aktualisierendes System zu aktivieren. Erst dann erhalten Sie Meldungen.

Vorgehensweise:

  1. Doppelklicken Sie in der Konsolenstruktur (links) auf dem MMC-Bildschirm auf den OpenView ManageX-Knoten, um ihn zu öffnen, und klicken Sie danach einmal auf die Erweiterung Meldungsleser, um sie auszuwählen.
  2. Klicken Sie mit der rechten Maustaste auf den Meldungsleser-Knoten, und wählen Sie dann im Kontextmenü Eigenschaften.
  3. Nehmen Sie im nun eingeblendeten Dialogfeld Eigenschaften alle gewünschten Einstellungen vor. Sie können jederzeit auf Übernehmen klicken, um die Änderungen wirksam werden zu lassen, ohne das Dialogfeld zu schließen.
    • Im Bereich Meldungsprotokoll können Sie mit Hilfe der Umlaufauswahlfelder bestimmen, wie viele Meldungen in der tabellarischen Anzeigeliste des Meldungslesers beibehalten werden. (Standardmäßig behält der Meldungsleser die letzten 250 Meldungen in einer rollbaren Anzeigeliste bei.)
    • Im Bereich Datum können Sie durch Markieren bzw. Aufheben der Markierung des Kontrollkästchens Datum anzeigen bestimmen, ob die Datumsspalte ein- oder ausgeblendet wird. Mit Hilfe der runden Optionsfelder in diesem Bereich können Sie zudem das Format der Datumsanzeige ändern. Sie haben die Auswahl zwischen Langes Datumsformat (z. B. Montag, Mai 11 1998) und Kurzes Datumsformat (z. B. 5/11/98). (Das kurze Datumsformat ist die Standardeinstellung.)
    • Im Bereich Uhrzeit können Sie durch Markieren bzw. Aufheben der Markierung des Kontrollkästchens Uhrzeit anzeigen bestimmen, ob die Zeitspalte ein- oder ausgeblendet wird. Mit Hilfe der runden Optionsfelder in diesem Bereich können Sie zudem das Format der Zeitanzeige von der Standardeinstellung Lokales Zeitformat (d. h. das Zeitformat für das in den Windows-Ländereinstellungen festgelegte Standardgebietsschema) auf das 24-Stunden-Format (23:28:00) ändern.

      Hinweis: Im 24-Stunden-Format ist das Sortieren der Meldungen nach Uhrzeit wahrscheinlich effizienter.
    • Der Bereich Konfigurationsdatei ist in dieser Version von ManageX nicht aktiv.
  4. Nachdem Sie alle Einstellungen vorgenommen haben, klicken Sie auf OK, damit die Änderungen wirksam werden, oder auf Abbrechen, um die vorherigen Anzeigeeigenschaften wiederherzustellen.
