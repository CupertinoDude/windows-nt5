Installation de ManageX

ManageX utilise la routine d'installation standard du fichier setup.exe.

Pour installer ManageX

  1. Ouvrez setup.exe.
  2. Le processus d'installation Install Shield démarre automatiquement. Pour une installation des plus simples, acceptez les défauts suivants proposés à l'écran :
    Ecran Réponse par défaut
    Bienvenue... Suivant
    Contrat de licence logiciel Oui
    Sélection du dossier d'installation Suivant
    (le défaut est C:\Program Files\HP OpenView ManageX)
    Sélection de dossier programme  Suivant (le défaut est HP OpenView ManageX)
    Copie de fichiers Suivant
  3. Lorsque le processus d'installation est terminé, un explorateur de web démarre et charge le fichier d'information de ManageX. Vérifiez ce fichier puis quittez l'explorateur.
  4. Dans la boîte de dialogue Installation terminée qui s'affiche, cliquez sur Terminer pour fermer la boîte et revenir au Bureau.
