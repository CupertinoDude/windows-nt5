Configuration des propriétés d'affichage des messages

Le volet de résultats du Lecteur de messages (à droite) affiche un tableau continuellement mis à jour contenant les messages reçus en provenance des autres machines faisant partie de votre domaine et sous-réseau. Vous pouvez configurer divers aspects du format et du contenu de l'affichage à l'aide de la commande Propriétés de l'extension.

ATTENTION :
Vous devez cliquer au moins une fois sur le Lecteur de messages à chaque fois que vous ouvrez une session avec ManageX de manière à activer son système de mise à jour dynamique. Vous ne pourrez pas recevoir de messages avant de l'avoir fait.

Pour configurer l'affichage des messages :

  1. Dans l'arborescence de console MMC (volet de gauche), double-cliquez sur le nœud OpenView ManageX pour l'ouvrir puis cliquez une fois pour sélectionner l'extension Lecteur de messages.
  2. A l'aide du bouton droit de la souris, cliquez sur le nœud Lecteur de messages et sélectionnez  Propriétés dans le menu de raccourci.
  3. Faites votre choix dans la boîte de dialogue Propriétés qui s'affiche. Vous pouvez cliquer sur Appliquer à tout moment pour activer vos modifications sans avoir à refermer la boîte de dialogue.
    • Dans la section Historique des messages, utilisez les boîtes de rotation pour sélectionner le nombre de messages à présenter sous forme de tableau dans le Lecteur de messages. (Le Lecteur de messages conserve par défaut les 250 derniers messages dans une liste d'affichage défilable.)
    • Dans la section Date, cochez ou enlevez la coche de la case Afficher la date pour afficher ou non la colonne de la date. Vous pouvez également utiliser les cases d'option de cette section pour spécifier l'affichage de la date, soit Date longue (par ex., Lundi 11 mai 1998) ou Date courte (par ex., 11/5/98). (La date courte est le défaut.)
    • Dans la section Heure, cochez ou enlevez la coche de la case Afficher l'heure pour afficher ou non la colonne de l'heure. Vous pouvez également utiliser les cases d'option de cette section pour modifier le format de l'affichage de l'heure, soit Format horaire local (le format horaire correspondant aux Paramètres régionaux de Windows sur votre système) ou Format horaire de 24 heures (23:28:00).

      Remarque : Le format horaire de 24 heures peut accroître l'efficacité du triage des messages par heure d'arrivée.
    • La section Fichier de configuration n'est pas active dans la présente version de ManageX
  4. Après avoir effectué vos choix, cliquez sur OK pour appliquer les modifications ou sur Annuler pour revenir aux propriétés d'affichage précédentes.
