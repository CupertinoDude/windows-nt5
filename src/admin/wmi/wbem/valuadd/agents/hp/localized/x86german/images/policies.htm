Richtlinien

Zu dieser Version von ManageX gehören zwei Richtlinien, die das NT-Ereignisprotokoll überwachen und entsprechende Benachrichtigungen zum ManageX-Meldungsleser weiterleiten:

  • Die Richtlinie NtCriticalEvents.mxw, die während der Installation aktiviert wird, sendet nur dann Meldungen, wenn als kritisch eingestufte Ereignisse auftreten.
  • Die Richtlinie NtAllEvents.mxw, die während der Installation deaktiviert wird, sendet beim Auftreten aller Ereignisse, die im NT-Ereignisprotokoll vermerkt sind, Meldungen.

Ändern des Richtlinienstatus

ManageX besitzt ein Befehlzeilen-Dienstprogramm, MxPolicyMgr, mit dem Sie den Standardstatus der beiden Richtlinien ändern können.

Aktivieren und Deaktivieren von Richtlinien

  1. Wechseln Sie nach der Befehlseingabeaufforderung in das ManageX-Installationsverzeichnis. Wenn Sie bei der Installation die Standardeinstellungen übernommen haben, handelt es sich hierbei um das Verzeichnis C:\Programmdateien\HP OpenView ManageX.
  2. Um die Richtlinien zu aktivieren, geben Sie die beiden folgenden Zeilen ein, und drücken Sie nach jeder Zeile die Eingabetaste. (Wenn Sie nur eine der Richtlinien aktivieren möchten, geben Sie nur die Zeile mit dem gewünschten Richtliniennamen ein.)

    MxPolicyMgr -enable Policies\NTCriticalEvents.mxw
    MxPolicyMgr -enable Policies\NTAllEvents.mxw
  3. Um die Richtlinien zu deaktivieren, geben Sie die beiden folgenden Zeilen ein, und drücken Sie nach jeder Zeile die Eingabetaste. (Wenn Sie nur eine der Richtlinien deaktivieren möchten, geben Sie nur die Zeile mit dem gewünschten Richtliniennamen ein.)

    MxPolicyMgr -disable Policies\NTCriticalEvents.mxw
    MxPolicyMgr -disable Policies\NTAllEvents.mxw
  4. Kehren Sie zu ManageX in der Microsoft Management Console™ zurück, und klicken Sie auf den Meldungsleser, um ihn zu aktivieren. Beim Auftreten der betreffenden im NT-Ereignisprotokoll vermerkten Ereignisse erhalten Sie nun eine entsprechende Meldung.
