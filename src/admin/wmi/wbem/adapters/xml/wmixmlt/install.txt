Here are the installation instructions for the XML Adapter.  There are 2 sections - one
dealing with the executable components, and the other with the DTDs.

Executables
===========

 1) Copy wmixmlt.dll and wmixmlt.tlb to the WBEM install directory

 2) Run regsvr32 on wmixmlt.dll


DTDs
====

 1) Ensure the subdirectory .\dtd of the wbem installation directory exists
 
 2) Copy the files ..\DTD\CIM20.DTD and ..\DTD\WMI20.DTD to that directory
