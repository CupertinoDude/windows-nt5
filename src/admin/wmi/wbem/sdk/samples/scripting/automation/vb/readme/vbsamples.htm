Windows Management Instrumentation SDK


Visual Basic Scripting Samples

 

Each of the subdirectories in this folder contains the project files required to build the sample in Visual Basic 6.0.

 

Any additional controls required are described below on a per-sample basis.  These controls are installed automatically as part of Visual Basic 6.0 Professional or Enterprise editions. 

 

NOTE:  It should be noted that these controls were not shipped with Visual Basic 5.0 (or as part of the Visual Basic 5.0 Service Packs) and therefore the samples will not work using Visual Basic 5.0.  Visual Basic 6.0 did ship with these controls, however, and the samples will work correctly for this version.

 

VBInstance

 

An instance property graphing application.  This sample should be run using 1024x768 (or higher) screen resolution.

 

Requires:  MSCHRT20.OCX, MSFLXGRD.OCX

 

VBLogDisk

 

View information about disks.

 

VBProcess

 

View information about processes.

 

Requires:  MSCOMCTL.OCX, MSCOMCT2.OCX

 

VBService

 

Allows services to be managed.  This sample should be run using 1024x768 (or higher) screen resolution.

 

Requires:  MSCOMCTL.OCX, MSCOMM32.OCX, COMDLG32.OCX, COMCT332.OCX, MSCOMCT2.OCX
