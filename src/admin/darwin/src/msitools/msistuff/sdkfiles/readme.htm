Configuration Utility for the Windows® Installer Internet Download Bootstrapper

Msistuff.exe is a configuration utility for Setup.exe, the bootstrap executable for Microsoft® Windows® Installer version 2.0. Msistuff provides the user with the ability to configure certain resources within setup.exe. Setup.exe uses the values of these resources to download and install Windows® Installer packages from the web.

Thus, using Msistuff.exe, anyone can quickly configure setup.exe to download and install a package of their choice from the web without modifying or recompiling setup.exe.

Building the sample

The sample can be built by running the following command from a Platform SDK build environment window: 

	nmake
In order to build the sample clean, the following commands should be used: 
	nmake clean
	nmake

Using Msistuff.exe

Msi.chm is the primary source of information about the Windows® Installer, and contains detailed reference material for developers. See the topics "Internet Download Bootstrapping" and "Msistuff.exe" for information on how to to use Msistuff.exe.

Limitations

This sample is not supported on 64-bit platforms. It is not supported on Windows 95/98/Me. It is only designed to work on 32-bit Windows NT based platforms.
