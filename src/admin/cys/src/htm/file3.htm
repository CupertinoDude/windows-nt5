Indexing Service
Users can search indexed files for specific words or characters.

Indexing Service catalogs the contents of the files in shared folders so that users can search them. Indexing Service is turned on by default when Whistler is installed.


 Because Indexing Service might slow the performance of this server, you should use it only if users frequently search the contents of files on this server.


Do you want to use Indexing Service on this server?

 Yes, leave Indexing Service on

 No, turn Indexing Service off
