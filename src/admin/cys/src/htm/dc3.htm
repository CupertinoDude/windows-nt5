Networking Properties: DHCP
DHCP assigns IP addresses to clients on a network.

On TCP/IP networks, client computers must be associated with IP addresses. Whistler uses DHCP to dynamically assign IP addresses to network computers and devices.

A DHCP scope uses a beginning IP address and an ending IP address to limit the range of IP addresses that this server can assign.
