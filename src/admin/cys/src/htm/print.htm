Printers and Printer Drivers
You can make printers and printer drivers available to clients on your network.

By using the Add Printer Wizard, you can install printers and their associated drivers for Windows 2000 and Whistler clients on your network. If some clients are running versions of Windows other than Windows 2000 or Whistler, use the Add Printer Driver Wizard to install the appropriate drivers for these clients.


For which clients do you want to install printers?
