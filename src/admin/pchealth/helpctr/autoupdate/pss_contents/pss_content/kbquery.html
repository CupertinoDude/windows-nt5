Querying the Microsoft Knowledge Base


This information describes how to more effectively query the Microsoft Knowledge Base by using keywords, including Microsoft product-specific keywords. By using keywords you may be able to find the article or articles that will help to resolve your issue or question.

When you perform a query, begin by using a product keyword that relates to your issue, such as "win95" for Microsoft Windows 95, "win98" for Windows 98, "outexw95" for Microsoft Outlook Express for Windows 95, and so on. You can then combine product keywords with other related keywords to help narrow your search, and locate the information you need. For example, if you want to find information about error messages when printing in Windows 98, you might query on:

win98 and kbprint and kberrmsg

This query would find all articles that contain all 3 of these keywords that relate to Windows 98, printing, and error messages. This type of query is called a Boolean query, and combines useful keywords to help you to search large amounts of information quickly. Note that not all articles will contain every appropriate keyword, so searches should be altered if you want to locate every possible piece of information. For example, if your issue appeared to relate to Windows 98, printing, and included an error message (as in the previous example), but you do not find the article you need, you may need to simplify your query. In the previous example, you could remove one of the keywords, and then perform the search again:

win98 and kbprint

Note that if you search by using a small amount of keywords, you will receive a large amount of articles, possibly making it difficult to find the article you are looking for. Because of this, you may need to perform searches more than once (making alterations as needed) to determine the best way to find the information you need. Other examples of queries include:

  • dun and msiew95 and kberrmsg

    This query will locate articles that contain these keywords, and that relate to Dial-Up Networking (DUN), Microsoft Internet Explorer for Windows 95 (MSIEW95), and an error message (kberrmsg).
  • multimon and win98 and kb3rdparty

    This query will locate articles that contain these keywords, and that relate to the multiple monitor feature (multimon), Windows 98 (Win98), and a third-party software program (kb3rdparty).
  • wincomm and kbhw and modem

    This query will locate articles that contain these keywords, and that relate to Windows communications topics (wincomm), computer hardware (kbhw), and modems.

    NOTE: "Modem" is not a keyword on the following list; in this example it is added to the search as an extra query word to help find any article that contains the word "modem." You can add additional words to your search if you think the word or words are likely to be in the article you want to find.

Commonly used keywords include:

  • win98 (Windows 98), win95 (Windows 95), wince (Windows CE)
  • msiew98 (Internet Explorer for Windows 98), msnetwork or msn (The Microsoft Network), winapm (Windows Advanced Power Management topics)
  • kbprint (printing topics), kberrmsg (error messages), kbsetup (topics related to installations), kbtool (topics related to tools or utilities), kbnetwork (topics related to networks)

The following section includes a more extensive list of most of the keywords used in the Microsoft Knowledge Base, including product keywords, issue keywords, and informational keywords.
