How to Create a Text File List of the Contents of a Folder

To create a text file that contains a list of the contents of a folder, create a file list using a command prompt:

  1. Click Start, point to Programs, and then click MS-DOS Prompt

2. At the command prompt, navigate to the drive that contains the folder whose contents you want to list. For example, if you want to create a text file that contains a list of the contents of a folder on drive C, type the following command and then press ENTER:

c:

3. At the command prompt, navigate to the folder whose contents you want to list. For example, if you want to create a text file that contains a list of the contents of the Windows folder on drive C, type the following commands, pressing ENTER after each command:

cd\

cd windows

4. Type the following command, and then press ENTER

dir > <filename>.txt

where <filename> is the name of the text file you are creating. For example, if you want to create a file named windowsfolderlist.txt, type the following command, pressing ENTER after the command:

dir > windowsfolderlist.txt

5. To return to Millennium, type the following command, pressing ENTER after the command"

exit

6. Use a text editor, such as WordPad, to view or print this file.

NOTE: The text file you create will be located in the folder you are in when you perform these steps. In the preceding example, the windowsfolderlist.txt file will be located in the Windows folder. If you cannot file the file you created, click Start, point to Search, click For Files or Folders, and type the name of the file you created in the "Search for files or folder named" box. Windows Explorer cannot export or print a list of the files contained in a folder.
