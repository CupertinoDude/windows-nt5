Minimum Hardware Requirements to Install Millennium

The minimum hardware requirements include:

  • Pentium 150
  • 32 megabytes (MB) of memory
  • 200 MB of hard disk space
  • CD-ROM or DVD-ROM drive
  • 3.5-inch high-density floppy disk drive
  • Video adapter and monitor that support VGA or higher resolution
  • Microsoft Mouse or compatible pointing device

The Windows Hardware Compatibility List (HCL) contains a list of computer systems and peripherals that are compatible with Windows. The HCL can be found on the Microsoft Windows Hardware Quality Labs Web site at:

http://www.microsoft.com/hwtest/
