Missing Log Off Option

When you try to log off your computer, the Log Off option is missing from the Start Menu. This behavior can occur when the Display Logoff check box has been cleared in the Taskbar properties.

To resolve this issue, select the Display Logoff check box:

  1. Click Start, point to Settings, and then click Taskbar & Start Menu.
  2. Click the Advanced tab, and then click to select Display Logoff on the Start Menu Settings box.
