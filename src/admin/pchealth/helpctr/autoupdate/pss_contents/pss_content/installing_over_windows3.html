Installing over Windows 3.x generates error

 

If you attempt to upgrade a Windows 3.x machine to Millennium you will be presented with the following error message:

Setup has detected that Windows 3.x is installed on this computer. Windows 3.x cannot be upgraded to this version of Windows. You must first install Windows 95 or Windows 98 and then install this version of Windows.

Click OK to exit Setup.

This behavior is by design. To work around this issue, install Millennium into a new directory.
