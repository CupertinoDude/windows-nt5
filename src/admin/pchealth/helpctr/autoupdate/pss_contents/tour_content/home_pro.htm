What Windows Millennium Provides

Windows Millennium provides simple installation of home networking on computers that run Microsoft Windows Millennium, Windows 98, and Windows 95. How easy? Just follow these three basic steps: 1. Set up your Windows Millennium computer. The Home Networking wizard makes sure your computer is networkready and sets up networking the way you want it. If you plan on sharing an Internet connection, it's best to set up your Windows Millennium computer for the connection. 2. Create a Home Networking Setup disk. It's easy to create this floppy disk. It contains the Home Networking wizard. 3. Set up each computer. Insert your Home Networking Setup disk in each computer you want to have in your home network, and run the wizard. When you want to change Home Networking settings, run the wizard again from Control Panel, and choose the options you want.
