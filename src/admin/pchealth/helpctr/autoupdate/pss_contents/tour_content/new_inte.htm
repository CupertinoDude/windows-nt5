Internet Migration

If you have more than one computer in your home, Windows Millennium Edition makes it easy for you to give each one the same Internet Explorer and Outlook Express account settings. This way, you don't have to go through the bother of setting up different accounts on each computer, or trying to figure out how to tie the computers into the same account.

Just run the Internet Connection wizard on the computer whose settings you want to duplicate, and then follow the instructions. Windows will copy the settings to a floppy disk, which you then insert into the computers you want to add the settings to.
