If this is your first time using Remote Assistance, please read the following information carefully.

Things to consider when using Remote Assistance
  • The helper should be a person you trust.
  • The helper must be using Whistler or later
  • You must be connected to the internet when your helper accepts your invitation.
  • If you are within a local area network, check with your network administrator before using Remote Assistance.
