Remote Assistance - Send an Invitation
Important Notes
  Before sending an invitation, be sure to read the information below.
 
  • For security reasons, make sure you only send invitations to people you trust.

  • Both you and your assistant must be running "Whistler" or later versions of Microsoft Windows.

  • To receive notification that your invitation has been accepted, you must be connected to the Internet. Learn about connection issues...

  • If your computer is connected to a corporate network, check with your network administrator before using Remote Assistance.
