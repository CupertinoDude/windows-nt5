Install and share Windows Help

Install, share, and quickly switch to other Help content. Begin by installing Help content from a CD or a shared directory for Windows XP Professional or related Windows server family products. After you have installed the new Help content, you can switch to that Help at anytime, or share it with other computers on your network.

Switch from one operating system's Help content to another
Install Help content from another Windows computer
Install Help content from a CD or disk image
Share your Help content with others on your network
Uninstall Help that you have installed from another Windows operating system


Note Installing and sharing Help content is available only for Windows XP and related Windows server operating systems. You cannot, for example, install Windows XP Help content on a computer running Windows 2000, or vice versa.
