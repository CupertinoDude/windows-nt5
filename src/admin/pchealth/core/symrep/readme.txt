To create a symbol repository on a server:
------------------------------------------

******   Note : Please run on a Win98 machine only   *******

Copy symrep.exe, makerep.bat and new98bins to a new directory on any Win98 
machine. Then run
cd <new dir>
makerep <symbol server share>

where <symbol server share> specifies the destination server share name 
where the symbol repository needs to be created.

This will copy available win98 and IE4.01 .SYM files to the new repository.
