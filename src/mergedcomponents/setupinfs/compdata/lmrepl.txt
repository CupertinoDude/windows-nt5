The software for Directory Replicator Service (LMRepl) is not compatible with this version of Microsoft Windows and will be disabled during the upgrade. Directory Replicator Service is replaced by the File Replication System in Windows XP.

For more information, see the "Determining Domain Upgrade Strategies" chapter in the Windows XP Deployment Planning Guide, which is part of the Windows XP Server Resource Kit and available on the Windows XP CD-ROM.

For a list of software supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com/isapi/redir.dll?prd=whistler&ar=help&sba=compatible.
