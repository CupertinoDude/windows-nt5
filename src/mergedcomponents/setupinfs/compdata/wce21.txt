Microsoft Windows XP Setup cannot determine the version of Microsoft Windows CE Services installed on this computer.
		 
Microsoft Windows CE Services version 2.2 is required to properly communicate with Windows XP. If you have an earlier version of Windows CE installed on your computer, you will not be able to use Windows CE products until you have upgraded to version 2.2. You can update Windows CE either before or after you upgrade to Windows XP. 
		 
For more information, visit Microsoft Windows CE Services 2.2 at the Microsoft Web site.
