The installed version of Microsoft Intellipoint Software is not compatible with Microsoft Windows XP and will be disabled during the upgrade. If you have version 3.1 or greater, you must reinstall the software after the upgrade has completed in order for it to function correctly in Windows XP.

Although having the software installed will not affect the setup process, it is recommended that you uninstall the software before continuing with the upgrade.

For more information about this software, or to obtain a version that runs on this version of Microsoft Windows, visit the Microsoft Web site at http://www.microsoft.com/mouse.

For a list of devices supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com.
