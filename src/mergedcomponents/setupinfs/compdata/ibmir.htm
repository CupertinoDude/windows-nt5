The currently installed driver for the Low Speed IRDA is not compatible with this version of Microsoft Windows and will be disabled during the upgrade. You should uninstall this device for a more complete solution.

To uninstall this device

  1. Insert the IBM IRDA Driver disk into drive A.
  2. Click Start, and then click Run.
  3. In Open, type A:\INSTALL.EXE, and then click OK.
  4. Click DeInstall.
  5. Click Add/Remove.
  6. Restart the computer, and then restart Windows XP Setup.

For more information about this driver, visit the manufacturer's Web site.

For a list of devices supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com.
