The currently installed driver for the Canon PowerShot ELPH S100 must be reinstalled after the Microsoft Windows upgrade.

For more information about this driver, visit the Canon Web site at http://go.microsoft.com/fwlink/?LinkId=34. Web addresses can change, so you might be unable to connect to this Web site.
 
For a list of devices supported by this version of Windows, see the list of compatible hardware on the Microsoft Web site at http://www.microsoft.com.
