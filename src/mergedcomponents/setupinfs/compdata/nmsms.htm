The software for Network Monitor Agent is not compatible with this version of Microsoft Windows and will be replaced with the Network Monitor Driver during the upgrade.

If you use version 1.x of Microsoft Network Monitor, which ships with Microsoft System Management Server, be aware that this program depends on the Network Monitor Agent for both local and remote captures and will not function correctly in Windows XP.

For more information about this software, or to obtain a version that runs on this version of Microsoft Windows, visit the Microsoft Systems Management Server home page at the Microsoft Web site.

For a list of software supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com.
