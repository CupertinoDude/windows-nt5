The software for Siemens MultiPath is not compatible with this version of Microsoft Windows and will be disabled during the upgrade. Windows XP drivers do not support multiple controllers for a single hard disk. 

You must uninstall this software and reconfigure your system before upgrading to Windows XP. 

1. Click Start, point to Settings, and then click Control Panel.
2. Double-click Add/Remove Programs, and then uninstall the MultiPath software.
3. When you are prompted to restart the system, do so.
4. Reconfigure your system with a single controller attached to each hard disk.
5. Restart the Windows XP installation.
 
For more information about this software, visit the manufacturer's Web site. 

For a list of software supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com/isapi/redir.dll?prd=whistler&ar=help&sba=compatible.
