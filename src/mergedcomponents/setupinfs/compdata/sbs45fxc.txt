Your existing Microsoft fax application will not work after upgrading to Windows XP. Uninstall this application using Add/Remove Programs in Control Panel, and then restart the upgrade. 

To use this fax application after upgrading to Windows XP, you need to download the updated version of the application. This updated version is available on http://go.microsoft.com/fwlink/?LinkId=818. You will need to reconfigure your existing fax settings for this version of the application. 

For a list of software supported by this version of Windows, see the Microsoft Windows XP Product Compatibility List at http://www.microsoft.com/windows/compatible.
