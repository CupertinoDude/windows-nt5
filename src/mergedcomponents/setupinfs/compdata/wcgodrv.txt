The currently installed drivers for the Creative Video Blaster WebCam Go and Go Plus cameras are not compatible with the Microsoft Windows XP upgrade process. The camera MUST BE UNPLUGGED during the upgrade.

For more information about these drivers, visit the manufacturer's Web site.

For a list of devices supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com/isapi/redir.dll?prd=whistler&ar=help&sba=compatible.
