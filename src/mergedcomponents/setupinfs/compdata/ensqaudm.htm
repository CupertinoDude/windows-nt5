The software for Ensoniq AudioPCI Mixer is not compatible with Microsoft Windows XP and will be disabled during the upgrade. It is recommended that you uninstall this software before continuing with the upgrade.

For more information about this software, or to obtain a version that runs under Windows XP, visit the Creative Technology Ltd. Web site. Web addresses can change, so you might be unable to connect to this Web site.

For a list of software supported by Windows XP, see the Microsoft Windows XP Product Compatibility List at the Microsoft Web site.
