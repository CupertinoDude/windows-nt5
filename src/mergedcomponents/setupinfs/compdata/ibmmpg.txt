The currently installed driver for the [MCI] MPEG-2 Decoder driver is not compatible with this version of Microsoft Windows and will be disabled during the upgrade. It is recommended that you uninstall this software before continuing with the upgrade.

To uninstall this device
	1. Click Start, point to Settings, and click Control Panel.
	2. Double-click Multimedia.
	3. On the Devices tab, double-click Media Control Devices.
	4. Click [MCI] IBM MPEG-2 Decoder.
	5. Click Remove, and then click Yes.
	6. Click OK, and then restart the computer.
	7. Restart Windows XP Setup.

For more information about this driver, visit the manufacturer's Web site.

For a list of devices supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com/isapi/redir.dll?prd=whistler&ar=help&sba=compatible.
