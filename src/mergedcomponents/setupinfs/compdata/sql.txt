The software for Microsoft SQL Server 6.5 Standard Edition and Microsoft SQL Server 6.5 Enterprise Edition is not compatible with this version of Microsoft Windows. You must install SQL Server 6.5 Service Pack 5 or later for these programs to run under Windows XP.

For more information about this software, or to obtain a version that runs on this version of Microsoft Windows, visit the Microsoft SQL Server page at the Microsoft Web site.

For a list of software supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com/isapi/redir.dll?prd=whistler&ar=help&sba=compatible.
