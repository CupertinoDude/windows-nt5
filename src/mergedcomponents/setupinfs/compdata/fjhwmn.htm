The Intel LANDesk Client Manager for Fujitsu FMV is not compatible with Windows XP. Make sure the software is uninstalled before Windows XP upgrade.

Follow these instructions to remove the Intel LANDesk Client Manager:
1. Click Start, point to Settings, and then click Control Panel.
2. Select Add/Remove Programs.
3. Select Intel LANDesk Client Manager, and then click Remove.
4. After restarting your machine and logging back on, run Windows XP Setup to install Windows XP on your machine.

For more information about this software, or to obtain a version that runs on this version of Microsoft Windows, visit the manufacturer's Web site.

For a list of software supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com.
