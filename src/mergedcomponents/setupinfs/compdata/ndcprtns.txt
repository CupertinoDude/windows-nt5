The currently installed configuration driver for the 802.11 wireless devices is not compatible with this version of Microsoft Windows and will be disabled during the upgrade.

The current configuration will be preserved. The configuration utility will no longer work. The configuration can be changed using the Advanced tab on the Properties page in the Device Manager.

Microsoft cannot determine the manufacturer for this device. For more information about this driver, visit the manufacturer's Web site.

For a list of devices supported by this version of Microsoft Windows, see the Microsoft Windows&nbsp;XP Compatibility List at http://www.microsoft.com/isapi/redir.dll?prd=whistler&ar=help&sba=compatible.
