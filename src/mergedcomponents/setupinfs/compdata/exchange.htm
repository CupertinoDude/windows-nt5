The software for Microsoft Exchange Server 5.5 Standard Edition and Microsoft Exchange Server 5.5 Enterprise Edition is not compatible with this version of Microsoft Windows. You must install Exchange Server 5.5 Service Pack 3 or later in order for these programs to run under Windows XP.

For more information about this software and to download Exchange Server 5.5 Service Pack 3 or later, see the Microsoft Exchange Server page at the Microsoft Web site. Web addresses can change, so you might be unable to connect to the Web site mentioned here.

For a list of software supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com.
