Microsoft Windows XP Setup has detected that the computer you are upgrading is running Terminal Server (formerly "Terminal Services in Application Server mode"). Terminal Server is not supported on Windows XP Server. Before you proceed with this installation, you must choose one of the following options.

  • Continue to use this computer as a Terminal Server.
    1. Cancel this upgrade.
    2. Install Windows XP Advanced Server. Terminal Server is also included as part of Windows XP Datacenter Server.
  • Use this computer for Remote Desktop for Administration (formerly "Terminal Services in Remote Administration mode"), which allows you to make up to two Remote connections to manage the server computer remotely.
    1. Cancel this upgrade.
    2. Go to Control Panel, double-click Add/Remove Programs, double-click Add/Remove Windows Components, and then click Next.
    3. In the Windows Components dialog box, choose Remote administration mode.
    4. After the wizard finishes, reboot the computer.
    5. Upgrade to Windows XP Server.
