Microsoft Windows XP Setup has detected an Iomega Zip drive connected to the parallel port on your computer. Some installed tools for the Iomega Zip drive will not work properly after upgrading to Windows XP if your Iomega Zip drive is connected to the parallel port on your computer.

Although Windows XP does contain embedded support for the Iomega Zip drive, the drivers for this device will be disabled during the upgrade. It is recommended that you uninstall the Iomega Zip drive tools before continuing.

For more information, visit the manufacturer's Web site.

For a list of devices supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com.
