The currently installed software for the IBM Rapid Access keyboard is not compatible with this version of Microsoft Windows and the needs to be uninstalled. 

To uninstall this software:

1. Click Start, point to Settings, click Control Panel, and then click Add/Remove Programs.
2. Select IBM Rapid Access III keyboard, and then click Change/remove.
3. Follow the instructions to remove the software.

For more information about this driver, visit the manufacturer's Web site. 

For a list of devices supported by this version of Windows, see the list of compatible hardware on the Microsoft Web site at http://www.microsoft.com/isapi/redir.dll?prd=whistler&ar=help&sba=compatible.
