Some versions of Olympus CAMEDIA SmartDisk Flash Path are incompatible with Windows. Setup is unable to determine if your version of CAMEDIA is compatible with the new Windows version.

Olympus CAMEDIA will be disabled by Windows setup. Following Windows setup, please re-install a version that is compatible with the new Windows version.
		 
For more information about this driver, contact the device manufacturer or visit the manufacturer's Web site.
