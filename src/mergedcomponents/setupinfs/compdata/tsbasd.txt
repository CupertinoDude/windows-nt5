The software for the Toshiba Automatic Shutdown Subsystem is not compatible with this version of Microsoft Windows and must be uninstalled before the upgrade can continue. 

To uninstall Toshiba Automatic Shutdown Subsystem:
1. Click Start, point to Programs, point to Accessories, and click Command Prompt.
2. Do one of the following:

	-If you use Toshiba GS series, type > SETUPASD REMOVE.
	-If you use Toshiba MAGNIA series, type > SETUPASD UNINSTALL [-NOISC]. Use the [-NOISC] option if you use Intel LDSM and ISC.
		 
For more information about this software, or to obtain a version that runs on this version of Microsoft Windows, visit the manufacturer's Web site. 
 
For a list of software supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com/isapi/redir.dll?prd=whistler&ar=help&sba=compatible.
