The port monitor Hewlett Packard Network Port is no longer available in this version of Microsoft Windows. If you are using this port monitor, check to see if the device can support TCP/IP printing and change the configuration settings for each printer queue after setup has completed. If you are unsure what protocol this device supports, visit the manufacturers Web site to obtain additional port monitor compatible offerings.

For a list of software supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com.
