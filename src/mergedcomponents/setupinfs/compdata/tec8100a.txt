The following program is not compatible with this version of Windows and will not run. It is suggested that it be uninstalled before continuing with setup.

To uninstall Toshiba IDE Driver Setup Utility:

1. Click Start, click Settings, click Control Panel, and then double-click Add/Remove Programs.
2. Select Toshiba IDE Driver Setup Utility.
3. Click Add/Remove.
4. Click Uninstall.
5. Click Yes.
6. Click Yes to reboot.

For more information about this software visit, the manufacturer's Web site.
