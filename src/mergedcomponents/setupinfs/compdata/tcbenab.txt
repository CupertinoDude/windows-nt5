Enabler for ToshibaCardBusLANCard(LANCB00A) on your system is not compatible with Windows XP and the most part of power management feature will be provided by Windows XP itself. Therefore this software must be uninstalled before the upgrade can continue.

To uninstall this software:

1. Click Start, and then click Run.
2. In Run, type regedt32>, and then click OK.
3. Select HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services.
4. Delete CBTSBKey.
5. After restarting your system and logging back on, run Windows XP Setup to install Windows XP on your system.

For more information about the extension of power management on this version of Windows, visit the Toshiba Web site.
