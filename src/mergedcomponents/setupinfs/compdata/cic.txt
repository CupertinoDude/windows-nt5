The Handwriter version 2.0 software by CIC is not compatible with Microsoft Windows XP.

The tablet will no longer function, and the following error message will appear each time the system is started:

	CIC
	Could Not Set Priority
	
This error does not prevent Windows XP from being installed, and you can uninstall the CIC Handwriter software after Windows XP Setup is complete.
		 
For more information about this software, visit the manufacturer's Web site at http://go.microsoft.com/fwlink/?linkid=387. Web addresses can change, so you might be unable to connect to this Web site.

For a list of software supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com/isapi/redir.dll?prd=whistler&ar=help&sba=compatible.
