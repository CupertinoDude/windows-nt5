The software for Directory Service Migration Utility (DSMU) is not compatible with this version of Microsoft Windows. It is recommended that you uninstall this software before continuing with the upgrade. If you do not uninstall DSMU before continuing, you will not be able to uninstall Gateway Service for NetWare (GSNW) or IPX without first uninstalling DSMU.

To uninstall DSMU:
1. Click Start, point to Settings, point to Control Panel, and double-click Add/Remove Programs.  
2. Click Add/Remove Windows Components.
3. Select Dsmigrate to uninstall.
4. Manually delete all files from \program files\dsmigrat.

For a list of software supported by this version of Microsoft Windows, see the Microsoft Windows XP Compatibility List at http://www.microsoft.com/isapi/redir.dll?prd=whistler&ar=help&sba=compatible.
