ActiveVRML Documentation

The Active Virtual Reality Modeling Language (ActiveVRML) is a modeling language for specifying interactive animations. This draft specification has these sections:

A Brief Introduction to ActiveVRML
Introduces ActiveVRML, explains the key concepts of the ActiveVRML programming model, and illustrates how to use the language in short programming examples.
ActiveVRML Tutorial
Contains tutorial based on a simplified solar system.
ActiveVRML Reference Manual
Defines the ActiveVRML keywords, expressions and types, listing the syntax and semantics of the language.
ActiveVRML Index
Contains an index to the ActiveVRML keywords.
