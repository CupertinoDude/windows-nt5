There are 5 main scenarios:

  • No Settings: If you enter an email address like "JoeUser@ucla.edu", no settings will be returned.  The AutoDiscovery page will time out and the user will go to the Protocol/ServerName page.

  • Settings Downloaded: If you enter an email address like "JoeUser@yahoo.com", settings will be returned.  The AutoDiscovery page will take the user to the DisplayName page.

  • Use Web Based Mail: If you enter an email address like "JoeUser@aol.com", settings will be returned that tell the user to use a web page to send and receive email.  The AutoDiscovery page will take the user to the UseWebBasedMail page which on Win2k will have hyperlinks to launch the web page.

  • See Web Page for Info: If you enter an email address like "JoeUser@webtv.net", settings will be returned that tell the user to go to a web page to find out information about email support.  The AutoDiscovery page will take the user to the ServerInfo page which on Win2k will have hyperlinks to launch the web page.
