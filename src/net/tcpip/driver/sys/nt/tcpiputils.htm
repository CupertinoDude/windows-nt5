TCP/IP Utilities Component Description

This component supports the following TCP/IP management and troubleshooting utilities:

Utility Description
Arp.exe IP-MAC address resolution protocol related utility
Hostname.exe Prints name of the current host
Ipconfig.exe Displays Transmission Control Protocol (TCP) configuration information and renew and release status of the Dynamic Host Configuration Protocol (DHCP) address
Iprip.exe Routing Information Protocol (RIP) utility.
Netstat.exe Displays statistics on TCP, User Datagram Protocol (UDP), and Internet Protocol (IP)
Pathping.exe Performs connectivity troubleshooting when provided with the path information
Ping.exe Internet Control Message Protocol (ICMP) echo-based utility used to check for remote connectivity
Rexec.exe Carries out commands on remote hosts running the Remote Execute (REXEC) service. REXEC authenticates the user name on the remote host before executing the specified command
Route.exe Allows the addition, deletion, modification, and display of route entries
Rsh.exe Carries out commands on remote hosts running the Remote Shell (RSH) service.
Tracert.exe Displays intermediate node traversed on the path to a destination

Component Configuration

There are no configuration requirements for this component.
