Description

 

This document provides late-breaking or other information supplementing the documentation for the Windows NT 4.0 Microsoft (R) Internet Authentication Service snap-in.

Installation instructions

 

Run iasnt4.exe from the ...\valueadd\msft\mgmt\ias directory to install Windows NT4.0 IAS snap-in.

To run the snap-in, start MMC, load Internet Authentication Service snap-in, specify the name of a machine running Windows NT4.0 IAS.

Requirements

 

The Windows NT 4.0 Internet Authentication Service snap-in should be used to configure Windows NT 4.0 Server or Windows NT4.0 MCIS Internet Authentication Service.
