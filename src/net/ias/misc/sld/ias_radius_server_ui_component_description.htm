IAS RADIUS Server UI Component Description

This Internet Authentication Service (IAS) Remote Authentication Dial-In User Service (RADIUS) Server user interface (UI) component contains the UI Microsoft Management Console (MMC) for an IAS snap-in. The IAS UI is divided into IAS snap-in and remote access policy extension snap-in. The remote access policy extension snap-in extends both the Routing and Remote Access Server (RRAS) and IAS snap-ins.

Configuration Requirements

There are no configuration requirements for this component.
