IAS RAP MMC Component Description

The Internet Authentication Service (IAS) remote access policies component contains the Microsoft Management Console (MMC) extension for the ISA RAP snap-in.

Remote access policies specify a set of conditions and connection settings that give network administrators more flexibility in authorizing connection attempts. The Windows 2000 Routing and Remote Access Service (RRAS) and Windows 2000 IAS both use remote access policies to determine whether to accept or reject connection attempts.

Configuration Requirements

No configuration is required to use this component.
