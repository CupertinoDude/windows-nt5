Internet Connection Sharing Discovery and Control Server Component Description

This component will enable home and small-office users to control and collect status information from an Internet Connection Sharing server. Client applications are available for Windows XP, Windows 98, and Windows ME. Using a client application, a user can connect or disconnect the Internet Connection Sharing server or gather connection statistics.

Component Configuration

The discovery and control server is part of Windows XP Internet Connection Sharing and is enabled when Internet Connection Sharing is enabled. The Advanced tab allows you to restrict users from controlling the shared connection. With this option enabled, users are restricted to discovery and statistics information.

The following table shows the configuration settings for this component.

Property Default setting Description
Allow network users to control or disable the shared Internet connection. Enabled Prevents Internet Connection Sharing clients from disconnecting the shared connection. This is beneficial for small office Connection Sharing setups.
