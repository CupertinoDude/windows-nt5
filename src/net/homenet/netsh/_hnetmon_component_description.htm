Hnetmon Component Description

This component currently implements the bridge context. In particular, this Netsh helper dynamic-link library (DLL) allows the user to view and set various flags for each bridged adapter. Due to the nature of the bridge, these flags can only be set by using the Hnetmon helper utility. If the network bridge is included in your configuration, the Hnetmon component should be included as well.

Component Configuration

There are no configuration requirements for this component.
