Windows Sockets Component Description

This component supports Windows Sockets. Windows Sockets provide applications with access to transport and network name resolution protocols.

Component Configuration

There are no configuration requirements for this component.

Special Notes

The default Microsoft transport provider installed as part of the Windows Sockets package supports all Transport Driver Interface (TDI) compliant transport drivers, such as TCP/IP, IPX/SPX.
