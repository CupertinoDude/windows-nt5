IPX, SPX, and NB Protocols Component Description

NwlnkIpx, NwlnkSpx, and NwlnkNb represent a suite of protocols designed to interoperate with Novell NetWare’s Internetwork Packet Exchange (IPX) standard. These are transport protocols that function as drivers in the Microsoft Windows NT kernel space. All three protocols, of which NwlnkSpx and NwlnkNb bind to NwlnkIpx, are typically installed or uninstalled as a set. Because these protocols are broadcast-based, installing them will increase the broadcast traffic on your network.

Configuring the Component

There are no configuration requirements for this component.
