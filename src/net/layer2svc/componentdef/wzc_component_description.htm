Wireless Zero Configuration Component Description

The Wireless Zero Configuration component provides automatic configuration for wireless IEEE 802.11 adapters and IEEE 802.1X Port based Network Access Control for both wireless IEEE 802.11 and wired Ethernet adapters.

Component Configuration

No configuration is required to use this component.

For More Information

The Wireless Zero Configuration component conforms to IEEE 802.11 and IEEE 802.1X (Draft 10) Standards. For more information on these topics, see the IEEE Web site.
