WINS Component Description

The WINS component is a high performance network service responsible for managing a flat name space in the network. WINS is built on top of the Netbios over TCPIP (NetBt) protocol. Network clients can register and query names from one or more WINS servers, WINS including a mechanism for replicating the name database accross the network.

Component Configuration

No configuration is required to use this component.

For More Information

The WINS component conforms to IETF RFC 883,1001 and 1002. For more information on these topics, see the Internet Engineering Task Force Web site.
