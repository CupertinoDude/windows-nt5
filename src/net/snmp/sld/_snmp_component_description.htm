SNMP Component Description

This component supports the Simple Network Management Protocol (SNMP). SNMP is an agent service that provides management systems with information about activities that occur at the Internet Protocol (IP) network layer. The SNMP agent monitors network traffic, and retrieves and updates local management information based on the requests from the SNMP manager. The agent also notifies registered managers with traps when significant events occur.

Component Configuration

By default, this component accepts SNMP Get, GetNext or GetBulk requests within the public community name. The SNMP snap-in configuration tool can be used to change the configuration of this component.
