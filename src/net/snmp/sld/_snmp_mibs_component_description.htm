SNMP MIBs Component Description

This component contains the Management Information Base (MIB) files for the Simple Network Management Protocol (SNMP). A MIB is a collection of formally described objects. Each object represents a particular type of information. This information is stored as a set of MIB variables. MIB objects can be accessed and managed with SNMP through a network management system. The management console performing SNMP queries on remote devices usually loads these MIB files.

Component Configuration

There are no configuration requirements for this component.
