IP1394 Component Description

The IP1394 component contains drivers that support networking using the IP v4 protocol over the IEEE 1394 bus specification. The drivers automatically detect the presence of other network-capable systems on a 1394 bus and sets up communication with them.

Component Configuration

No configuration is required to use this component.

For More Information

The IP1394 component conforms to IETF RFC 2734. For more information on these topics, see the Internet Engineering Task Force Web site.
