RTC Component Description

The Real-Time Communications (RTC) User Agent component is an application programming interface (API) based on the component object model (COM) that enables developers to implement communications applications using the Session Invite Protocol (SIP). The API supports desktop computer-to-desktop computer, desktop computer-to-phone, and phone-to-phone calls, which may include audio and video streaming, instant messaging, application sharing, and presence capabilities.

Component Configuration

There are no configuration requirements for this component.
