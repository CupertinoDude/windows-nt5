HID Phone TSP Component Description

The Human Interface Devices (HID) Phone Telephony Service Provider (TSP) is a Telephony API (TAPI) TSP that enables the use of HID phone handset devices connected through a universal serial bus. These phones are available to applications through the TAPI 2.x and TAPI 3.1 application programming interfaces (APIs).

Component Configuration

There are no configuration requirements for this component.
