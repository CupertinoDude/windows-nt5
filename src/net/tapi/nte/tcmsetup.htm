Telephony Client Setup Utility Component Description

This component contains the tcmsetup.exe file, which is a command-line utility that allows administrators to configure the client computer to use a specific remote Telephony server. A remote Telephony Service Provider (TSP) must be installed on the client machine.

This utility should only be used to configure Windows XP client computers to point to a Windows NT4 or Windows 2000 Telephony server. Windows XP Telephony servers will be discovered automatically from Active Directory directory services.

Component Configuration

There are no configuration requirements for this component.
