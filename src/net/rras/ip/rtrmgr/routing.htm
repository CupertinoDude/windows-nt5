Routing Component Description

Routing provides the infrastructure to configure a server to be a unicast or multicast router. The unicast router forwards unicast traffic towards the destination based on the unicast routing table, which is created by running a unicast routing protocol like Open Shortest Path First (OSPF) or Routing Information Protocol (RIP). The multicast router includes Internet Group Membership Protocol (IGMP) router and Proxy components enabling multicast forwarding to a network. It includes the Dynamic Host Configuration Protocol (DHCP) relay agent and RIP for Internetwork Packet Exchange (IPX).

Component Configuration

No dynamic Hypertext Markup Language (DHTML) user interface (UI) is present to configure the component.

Special Notes

This component is dependent upon the Dialup_Server, Netsh_Interface, Netsh_Routing, and Simple Network Management Protocol (SNMP) components.
