Description

 

This document provides late-breaking or other information that supplements the documentation for the Microsoft® Phone Book Administrator tool.

Installation instructions

 

To install Phone Book Administrator, run pbainst.exe from the VALUEADD\MSFT\MGMT\PBA directory on your Microsoft Windows XP Professional CD.

Requirements

 

The Phone Book Administrator tool requires the Microsoft® Windows XP Professional operating system.

Limitations

 

1.        You can post phone books created with this version of Phone Book Administrator to servers running older versions of the Phone Book Service. However, you cannot run an older version of Phone Book Administrator on a computer running Windows XP because of incompatibilities in database libraries. If a phone book file has been edited with this version of Phone Book Administrator, it cannot be edited with older versions of Phone Book Administrator.

2.        From the command line, you cannot import files whose names contain apostrophes.

3.        If you delete a phone book using Phone Book Administrator, the server is no longer notified of the deletion. See the Connection Point Services documentation for details on how to fully delete your phone books.

4.        Phone book databases are encrypted in this version of Phone Book Administrator. You cannot open these databases in Microsoft Access or any other database program.
