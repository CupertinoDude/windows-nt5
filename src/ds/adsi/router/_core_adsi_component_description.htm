Core ADSI Component Description

This component provides the basic functionality for the Active Directory Service Interfaces (ADSI). It is required for other ADSI provider components, such as the Lightweight Directory Access Protocol (LDAP) provider, the Microsoft Windows NT provider, Novell NetWare Directory Services (NDS) providers, and NWCOMPAT providers. Core ADSI will route any requests to the corresponding provider according to the path it is provided.

Component Configuration

There are no configuration requirements for this component.

For More Information

Documentation for ADSI can be found in the Platform SDK and the Directory Services section of the MSDN Library.
