ADSI NDS Provider Component Description

This component contains a provider for Active Directory Service Interfaces (ADSI) that permits access to Novell Directory Services (NDS) directories. With this component, you can search and modify NDS directories using Component Object Model (COM) interfaces exported by ADSI. These interfaces are usable from both Automation-compatible languages, such as Microsoft Visual Basic, Scripting Edition (VBScript) and vtable-based languages, such as Microsoft Visual C++.

This component uses Microsoft Client Services for NetWare to access the NDS servers. It is not compatible with third-party NetWare clients.

Component Configuration

There are no configuration parameters for this component.

For More Information

ADSI is documented in the Platform SDK and the MSDN Library, in the Directory Services section.
