This dir contains the sources for the NT5 multimaster file replication service.

_idl	RPC interface descriptions
admin	admin support functions
comm	communication DLL interface support
dblayer database interface functions (from Dir Service)
debexts	debugger extensions
doc	design docs
        doc\jetdocs - files copied from \\iyf\jetdrop\docs
inc	header files
repl	data move engine
setup	setup/install functions,files and scripts
test	test programs
util	common utility functions
xlib    external libraries (e.g. JET) for linking.
