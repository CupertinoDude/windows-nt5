NetJoin Component Description

NetLogon and NetJoin are required services to manage a computer's connection to a Windows domain.

Netjoin is responsible for joining a computer to a domain. This component also renames the computer on the joined domain.

Netlogon performs pass-through authentication that might occur during NTLM authentication. Netlogon performs maintenance for machine accounts on domain members, such as machine password change. Netlogon hosts the domain controller (DC) discovery component and maintains the DC discovery cache accessible for all components in the system. Netlogon on a Microsoft Windows 2000 primary domain controller (PDC) performs Security Account Manager (SAM) database replication to down-level Microsoft Windows NT 4.0 backup domain controllers (BDCs).

Component Configuration

There are no configuration requirements for this component.
