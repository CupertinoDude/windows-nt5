RunAs Component Description

The RunAs component allows you to run interactive applications as different security principals, or in a restricted security context. The RunAs component allows you to specify your credentials using either a user name and password, or a smart card and personal identification number (pin). This component also gives you the option to persist the credentials associated with a security principal, so that subsequent executions in the context of this principal will not require you to re-enter credentials.

The RunAs component is invoked using the command-line tool runas.exe.

Component Configuration

There are no configuration requirements for this component.
