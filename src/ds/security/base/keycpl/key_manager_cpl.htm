Component Description

Keymgr.cpl is a control panel applet which makes available the services of keymgr.dll for the server versions of Windows.  Workstation versions of Windows can reach the services of keymgr.dll by means of nusrmgr.cpl.

This component should be installed for a server version of Windows if it will be necessary to present the user interface for manipulating the Credential Manager (Key Manager applet).
