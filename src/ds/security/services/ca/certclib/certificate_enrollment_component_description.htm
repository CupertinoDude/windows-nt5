Certificate Enrollment Component Description

The certificate request client and certificate auto enrollment component allows you to request certificates, find certification authority (CA) on the network, and automatically keep certificates current.

Component Configuration

You must use auto enrollment on a domain that is directed through settings in the Microsoft Active Directory™ directory service. You can modify these settings through the Microsoft Management Console (MMC) snap-in tool, called Certificate Templates. No additional configuration is required if the certificate request client application programming interfaces (APIs) are used.

For More Information
