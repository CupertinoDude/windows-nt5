Smart Card CSP Component Description

The Smart Card cryptographic service provider (CSP) component supports features such as smart card logon and secure e-mail. Smart cards must be capable of certain RSA public key cryptographic operations. These functions are exposed using the CryptoAPI and specifically through a CSP. Typically, each type of smart card requires a CSP , which is provided by the card vendor.

Windows XP includes CSPs to support smart cards from the following vendors: Schlumberger, Gemplus, and Infineon.

Component Configuration

There are no configuration requirements for this component.
