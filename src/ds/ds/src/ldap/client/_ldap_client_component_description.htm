LDAP Client Component Description

This component supplies the Lightweight Directory Access Protocol (LDAP) client component. This client-side component exposes a C application programming interface (API) for performing operations, such as searching and adding objects, against a LDAP directory server, such as Microsoft Active Directory. This component supports the LDAP version 2 and version 3 protocols.

The LDAP provider for Active Directory Service Interfaces (ADSI) is built on top of this component.

Component Configuration

There are no configuration parameters associated with this component.

For More Information

Information about this component can be found under Lightweight Directory Access Protocol API in the Directory Services section of the Platform SDK or the MSDN Library. Additional information about the LDAP protocol can be found in Request For Comments (RFC) 2251.
