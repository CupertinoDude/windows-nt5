CSV Directory Exchange Utility Component Description

The comma-separated value (CSV) Directory Exchange Utility component contains a command-line utility, csvde.exe. Csvde.exe supports the bulk importing and exporting of directory information to and from a Lightweight Directory Access Protocol (LDAP) directory, such as the Microsoft Active Directory. This utility reads and writes data in a CSV file format. It is useful for administrators who need to move large amounts of data from one directory to another. This utility is also useful for viewing directory information in applications, which are capable of loading comma-separated value files, such as spreadsheets or databases, but which are not capable of directly accessing LDAP directories.

Component Configuration

There are no configuration parameters for this component.

Special Notes

Besides the file format used, csvde.exe and ldifde.exe differ in the types of operations they support during import. Csvde.exe can only be used to bulk-load the directory by creating new objects. Ldifde.exe can also be used to perform operations on preexisting objects, such as deleting them, modifying their attributes, or renaming them.
