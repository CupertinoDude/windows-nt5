Security Account Manager Client Description

The Security Account Manager (SAM) client is responsible for providing services to the NetUser Win32 application programming interface (API) and internal operating system (OS) components so that administrative tasks can be performed on the security principals that are maintained by SAM. The following are examples of such tasks: set password, change password, create user, create group, set description, display users, and so on.

Component Configuration

There are no configuration requirements for this component.
